レスポンシブデザイン
さまざまな画面サイズ、解像度などに合わせて スタイルを変更できるようにする一連の実践です。


name="viewport" content="width=device-width, initial-scale=1.0"

はじめに、HTMLにレスポンシブデザインを実践するため、HEAD要素内にビューポイントを設定するメタ要素を記述します。


~~~


レスポンシブデザインは、画面が特定のサイズ（または特定の解像度）より大きいかどうかによって、適用するCSSを設定することです。


スクリーンサイズによってリンクするCSSを区別する

rel="stylesheet" href="mobile.css" media="screen and (max-width: 550px)"

mobile.css を、最大 550px の横幅 のウインドウサイズを持つブラウザーにリンクする

rel="stylesheet" href="pc.css" media="screen and (min-width: 550px)"

pc.css を、最小 550px の横幅 のウインドウサイズを持つブラウザーにリンクする


~~~


CSSに、複数のスクリーンサイズに適用するCSSを記述する


@media screen and (max-width: 数値) {
  CSSルールセット
}

@media screen and (min-width: 数値) {
  CSSルールセット
}

portraitは、スクリーン（ビューポート）が 縦長 高さが幅よりも大きいか等しい状態 のブラウザーを指定するキーワード値
landscapeは、スクリーン（ビューポート）が 横長 幅が高さよりも大きい状態 のブラウザーを指定するキーワード値

rel="stylesheet" href="landscape.css" media="(orientation: landscape)"
rel="stylesheet" href="portrait.css" media="(orientation: portrait)"

or


@media (orientation: landscape) {
  CSSルールセット
}

@media (orientation: portrait) {
  CSSルールセット
}

~~~


JavaScript で UserAgent を判定し、モバイル端末にのみ処理を実行する


if (/Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent)) 
{
  処理を記述する
}
