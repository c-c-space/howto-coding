Web Storage には、2 種類の仕組みがあります。

sessionStorage
ページのセッション中のみデータを保存します。（データがサーバーに転送されることはありません。）

localStorage
有効期限なしでデータを保存します。（ブラウザーを閉じたり再び開いたりしてもデータは持続します。）

ストレージオブジェクトがすでに存在しているか (すなわち、過去にページへアクセスしていたか) を確認します。

// 使用する Web Storage の種類を設定
const storage = localStorage;

// Web Storage に key アイテムが存在するかを確認
if(!storage.getItem('key')) {
  populateStorage();
} else {
  getStorage();
}

アイテムが存在しなければ 既存の値をストレージへ追加するために populateStorage() を実行し、 値が存在する場合は、値を取得するために getStorage() を実行します。

populateStorage()

// Web Storage にデータを保存する
function populateStorage() {
  storage.setItem('key', 'value');
}

getStorage()

// Web Storage に保存したデータを取得する
function getStorage() {
  let data = storage.getItem('key');
}

Web Storage にアクセスし、.removeItem() を利用してデータを削除します。

// Web Storage に保存したデータを削除する
storage.removeItem('key');

// Web Storage に保存したすべてのデータを削除する
storage.clear();
