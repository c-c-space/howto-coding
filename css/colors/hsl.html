hsl() 関数表記
hsl(hue saturation lightness) または hsl(hue saturation lightness / alpha)


hsl() 関数は、 色相(hue) ・ 彩度(saturation) ・ 輝度(lightness) の値を取り、約 1670 万色を表現します。


~~~


色相(hue) 色のベースとなるシェード。
0 から 360 の値を取り、色相環の角度を表します。 


background: conic-gradient(
  hsl(0, 100%, 50%),
  hsl(45, 100%, 50%),
  hsl(90, 100%, 50%),
  hsl(135, 100%, 50%),
  hsl(180, 100%, 50%),
  hsl(225, 100%, 50%),
  hsl(270, 100%, 50%),
  hsl(315, 100%, 50%),
  hsl(360, 100%, 50%)
);

背景のグラデーションカラーは、0(赤) から 360(赤) まで右回りに 色相(hue) の値を表しています。
赤(0/360) 黄(60) 緑(120) シアン(180) 青(240) マゼンタ(300)


~~~


彩度(saturation) 色がどれだけ飽和しているか?
0 から 100% の値を取り、0 は色がなく (グレーに見える)、100% はフルカラーの飽和となります。


輝度(lightness) 色がどれだけ明るいのか?
0 から 100% の値を取り、0 は明度がなく (完全な黒)、100% はフルの明度です (完全な白)。


~~~


透明度(alpha) hsl(hue saturation lightness / alpha)


0 から 1 までの数値、 または 1 から 100% までのパーセント値 で色の透明度を指定します。
1(100%) は完全に不透明で、 0(0%) は完全に透明であることを意味します。
