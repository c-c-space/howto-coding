16進表記
#RGB[A] または #RRGGBB[AA]


16進数の値には、 # に続く 6個（3個でも可）の 値 0 ~ f を使用します。


~~~


#RGB


R (赤) G (緑) B (青) に、0 から ff までの数値を指定し、それぞれの色の濃さを表します。
数値が 1つの場合は 同じ数値を複製し、2桁の数値を表します。


0 は その色 を使用しないことを意味し、f(ff) になるほど 色が濃くなります。
#000000(#000) は 黒 ・ #ffffff(#fff) は 白 です。



#RGBA


A (AA) に、0 から ff までの数値を指定し、色の透明度を表します。
数値が 1つの場合は 同じ数値を複製し、2桁の数値を表します。


0 は 完全に透明な色で、 f(ff) は 完全に不透明な色です。



#f07
#F07
#ff0077
#FF0077

#f070f
#F070F
#ff0077FF
#FF0077ff


以上のコードはすべて、 赤 f(ff)（最大の濃さ）・緑 0(00)（使用しない）・青 7(77)（7の濃さ）の完全に不透明 f(ff) な色 を指す値です。 色コードの中の大文字小文字は、区別されません。



~~~


rgb[a]() 関数記法
rgb(R, G, B) または rgba(R, G, B, A)


rgb() 関数記法の値には、 R, G, B を示す 0 ~ 255 の数値を使用します。


透明度を含む rgba() 関数記法の値には、 R, G, B, の値に加えて、 透明度を示す A の値に、 0 ~ 1 の数値を使用します。



rgb (R, G, B)


R (赤) G (緑) B (青) の値が、0 だとその色を使用しないことを意味し、255 になるほど色が濃くなります。

rgb(0, 0, 0) 黒 ・ rgb(255, 255, 255) は 白 です。



rgba (R, G, B, A)


A (透明度) 0 は 完全に透明な色で、1 は 完全に不透明な色です。
数値は、 0.00001 や 0.99 のように、コンマ単位で指定できます。



rgb(255, 0, 127.5)
rgba(255, 0, 127.5, 1)


以上のコードはすべて、 赤 255（最大の濃さ）・緑 0（使用しない）・青 127.5（50%の濃さ）の完全に不透明 1 な色 を指す値です。
