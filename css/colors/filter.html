filter:
blur(値)
brightness(値)
contrast(値)
drop-shadow(値)
grayscale(値)
hue-rotate(値)
invert(値)
opacity(値)
saturate(値)
sepia(値)


ぼかしや色変化などのグラフィック効果を要素に適用する CSSプロパティ
複数のフィルター を使用する場合は、プロパティ値に 関数と値 を追加します。


~~~


blur()
px・rem・vw・vhなどの値でぼかしを適用 (パーセント値は不可)


brightness()
数値・パーセント値で輝度を指定 (0%・0 は 完全に黒、100%・1 は 効果なし)


contrast()
数値・パーセント値でコントラストを調整 (0%・0 は 完全な灰色、100%・1 は 効果なし)


drop-shadow()
数値でドロップシャドウ効果を適用 (offset-x offset-y blur-radius color)


grayscale()
数値・パーセント値でグレースケールに変換 (0%・0 は 効果なし、100%・1 は 完全にグレースケール)


hue-rotate()
deg・turnなどの値で色相を回転 (0deg・0turn は 効果なし、90deg は 90 度の回転、.5turn は 180 度の回転)


invert()
数値・パーセント値で色サンプルを反転(0%・0 は 効果なし、100%・1 は 完全に反転)


opacity()
数値・パーセント値で透過度を適用(0%・0 は 完全に透明、100%・1 は 効果なし)


saturate()
数値・パーセント値で彩度を変化(0%・0 は 無彩色、100%・1 は 効果なし)


sepia()
数値・パーセント値でセピア調に変化(0%・0 は 効果なし、100%・1 は セピア調)
