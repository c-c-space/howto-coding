ボックスモデル

CSS には HTML 要素 を箱として扱う ボックスモデル という概念があり、ボックスには ブロックボックス と インラインボックス の 2種類があります。

ページ上のほとんどの HTML 要素は、他の箱の上に置かれた箱と考えることができます。


ブロックボックス
display: block;

ブロックボックスには改行（マージン）が入り、ボックスは上下に分割されます。

通常、ブロックボックスの幅は親要素と同じ値で、高さはコンテンツの内容によって変動します。


インラインボックス
display: inline;

インラインボックスは改行されず、デフォルトでは 左から右 に 小さなボックス として分割されます。

通常、インラインボックスは、ブロックボックスの中で使用する、「」のような役割の 小さな部分 に使用します。


このページにある要素の内、h1, p, nav, section などが ブロックボックス、 a, button, span などが インラインボックス です。



ボックスの構成

各ボックスは 4つの領域 で構成されます。


コンテンツ領域
box-sizing: content-box;

box-sizing の プロパティ値 content-box は、ボックスの幅・高さにコンテンツ領域のみを含む設定です。
（詳しくは 幅と高さ を参照ください。）

文字・画像・動画など、「実際の」コンテンツを表示するコンテンツ領域がボックスの中心（箱の中身）です。


境界領域
box-sizing: border-box;
border: 幅 種類 色;

box-sizing の プロパティ値 border-box は、ボックスの幅・高さに境界領域まで（コンテンツ＋パディング＋境界領域）を含む設定です。

ボックスの内と外を区切る境界（箱の外装）が境界領域です。
CSSプロパティ border で、境界線の 色・種類・幅など をスタイリングできます。 


/* 種類 */
border: solid;

/* 幅 | 種類 */
border: 2px dotted;

/* 種類 | 色 */
border: outset #f33;

/* 幅 | 種類 | 色 */
border: medium dashed green;

境界領域に関連するCSSプロパティには、境界に画像を置き換える border-image ・ 境界の外側の角を丸める border-radius などがあります。
（詳しくは 枠線 を参照ください。）


パディング領域
padding: パディング領域;

ボックスのコンテンツ（中身）と 境界（外装）の間の空間（余白）が、パディング領域です。
CSSプロパティ padding で、全四辺のパディング領域をスタイリングできます。 


/* 四辺すべてに適用 */
padding: 1em;

/* 上下 | 左右 */
padding: 5% 10%;

/* 上 | 左右 | 下 */
padding: 1em 2em 2em;

/* 上 | 右 | 下 | 左 */
padding: 5px 1em 0 1em;

ボックスに background プロパティで背景をスタイリングした場合、パディング領域まで背景が適用されます。
（詳しくは 背景 を参照ください。）


マージン領域
margin: マージン領域;

マージン領域は、境界領域の外側にある、ボックスとボックスの間の空白です。
CSSプロパティ margin で、全四辺のマージン領域をスタイリングできます。 


/* 四辺すべてに適用 */
margin: 1em;

/* 上下 | 左右 */
margin: 5% 10%;

/* 上 | 左右 | 下 */
margin: 1em 2em 2em;

/* 上 | 右 | 下 | 左 */
margin: 5px 1em 0 1em;

マージンの値は、ボックスサイズに含まれることはなく、 auto を使用して、要素を中央揃えすることができます。
（詳しくは 幅と高さ を参照ください。）
