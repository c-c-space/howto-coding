box-shadow
ボックスの周囲にシャドウ効果を追加するCSSプロパティ 


/* キーワード値 */
box-shadow: none;

/* 上下 | 左右 | 色 */
box-shadow: 10px 10px silver;

/* 上下 | 左右 | ぼかし | 色 */
box-shadow: 10px 10px 5px silver;

/* 上下 | 左右 | ぼかし | 影の拡大 | 色 */
box-shadow: 5px 5px 5px 10px #eee;

/* inset (ボックスの内に影を描画) | 上下 | 左右 | 色 */
box-shadow: inset 2.5vh 2.5vw #000;

/* 複数の影はカンマで区切る */
box-shadow: 3px 3px red, -1rem 0 0.5vh 0.5vw olive;
