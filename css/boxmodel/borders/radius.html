border-radius
境界の外側の角を丸めるCSSプロパティ 


/* 全4角の丸みを設定 */
border-radius: 50px;

/* 左上と右下 | 右上と左下 */
border-radius: 25% 50%;

/* 左上 | 右上と左下 | 右下 */
border-radius: 5rem 1rem 10%;

/* 左上 | 右上 | 右下 | 左下 */
border-radius: 10vw 30vh 50vw 70vh;

/* 楕円の横の半径と縦の半径をスラッシュ（/）で区切る */
border-radius: 5rem / 25%;

/* (最初の半径の値) 左上 | 右上 | 右下 | 左下 / 左上 | 右上 | 右下 | 左下 */
border-radius: 25vw 1rem 10vh 1rem / 1rem 25vw 1rem 100px;

丸みの値は、px・rem・vw・vhなどの数値か、%で指定します。


~~~


border-image
境界領域に画像を描画するCSSプロパティ

border が none または 0 の場合、境界画像を描画しないブラウザーもあります。


/* 画像URL | スライスの指定 */
border-image: url() 25%;

/* グラデーション | スライスの指定 (上下 | 左右) */
border-image: gradient() 10% 20%;

/* グラデーション | スライスの指定 (上 | 右 | 下 | 左 | 真ん中) */
border-image: gradient() 10 20 15 25 fill;

border-image には border-radius が適用されません。
