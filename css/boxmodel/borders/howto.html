境界線 （境界領域）

border: スタイル 幅 色;
要素に境界線（境界領域）を追加するとき、CSSプロパティ border を使えば、スタイル・幅・色を一括設定できます。


~~~


線のスタイルを設定するキーワード値

none 境界線を表示しません
solid 1 本の直線（既定値）
dotted 連続した丸い点
dashed 連続したダッシュ・線分
double 2 本の直線・二重線
groove 凹んだように見える立体的な線
ridge 出っ張ったように見える立体的な線
inset 要素が埋め込まれて見える立体的な線
outset 要素が出っ張って見える立体的な線


~~~


線の幅を設定する値

thin ≤ medium ≤ thick
それぞれのキーワードで示される正確な太さは定義されていません。


数値px 画面表示における伝統的な単位
数値rem デフォルトで設定されたフォントサイズを 1 とする単位
数値vw ウインドウの幅 の 1% を 1vw とする相対的な単位
数値vh ウインドウの高さ の 1% を 1vh 相対的な単位


~~~


線の色を設定する値

詳しくは 色 を参照ください。

~~~


輪郭線
outline: スタイル 幅 色;

境界線と輪郭線はよく似ています。線のスタイリング方法は border と同じです。

輪郭線は、境界領域の外側に描画され、ブロックの領域を占有しません。
