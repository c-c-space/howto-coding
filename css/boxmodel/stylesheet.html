ブロック要素とインライン要素

HTML の要素には知っておくべき 2 つの重要なカテゴリー ブロックレベル要素 と インライン要素 があります。

この分類は、HTML Living Standard (HTML5以降) では廃止になりましたが、 ページレイアウトを作成するための重要なコツとなります。

ブロックレベル要素
見出し（h1 ~ h6）・段落（p）・リスト項目（ul ol dl li）・罫線（hr）・div など

ブロックレベル要素は、CSS の display が block に設定されています。

また、 body にもデフォルトで、マージンが設定されています。 


* {
  border: none;
  padding: 0;
  margin: 0;
}

以上の CSS ルールセット で、 すべての種類の要素 の 境界・パディング・マージン を なし(0) に スタイリングします。


インライン要素
リンク（a）・文字装飾（b em i u など）・span など

インライン要素は、CSS の display が inline に設定されています。 


nav a,
span {
  display: inline-block;
}

display の プロパティ値を inline-block に設定すると、 block と inline の特性を混在させ、 インライン要素 のように 改行を作らないまま ブロックレベル要素 のように パディング・マージンを指定できます。
ここでは nav 要素内の a 要素 と span 要素 のディプレイ値 を inline-block に設定し、パディング・マージンを操作します。


~~~~~


/* style.css */


* {
  border: none;
  padding: 0;
  margin: 0;
}

html {
  height: -webkit-fill-available;
}

body {
  background: #000;
  min-height: -webkit-fill-available;
}

body,
a {
  color: #fff;
}

nav a,
span {
  display: inline-block;
}

body,
h1,
.special {
  border: dashed 1px #fff;
}
