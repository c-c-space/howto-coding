height

height: 値; は、要素の幅を設定する宣言(CSS プロパティ)です。
最小幅を設定する min-height と 最大幅を設定する max-height は、 height を上書きします。 


/* length 値 */
height: 250px;
height: 50vw;
height: 75vh;

/* キーワード値 */
height: -webkit-fill-available;

length 値
height: 50vw;
height: 75vh;


CSS には、長さ (length) の値を表わす単位が数多くあります。


vw は、ページを閲覧するウインドウの横幅によって変動する 相対的な値です。
例えば、50vw は ウインドウの横幅の 半分 の寸法です。


vh は、ページを閲覧するウインドウの高さによって変動する 相対的な値です。
例えば、75vh は ウインドウの高さの 75% の寸法です。



CSS fix for 100vh in mobile WebKit
height: -webkit-fill-available;


html {
  height: -webkit-fill-available;
}

body {
  min-height: -webkit-fill-available;
}

main {
  max-height: -webkit-fill-available;
}

モバイル端末用:フルサイズの高さを設定するウェブキットの値

高さの値に -webkit-fill-available を 設定すると モバイル用ブラウザーのナビゲーションバーを除く、ウインドウの高さのフルサイズを指定できます。
モバイル端末ではブラウザーのナビゲーションバーがページ下部を覆い隠すため、 100vh が うまく動作しません。

左右のはみ出しは overflow-x、 上下のはみ出しは overflow-y で個別に設定
overflow: scroll hidden; のように、 overflow-x(左右) overflow-y(上下) の値を設定し、一括指定も可能です。
