Width

width: 値; は、要素の幅を設定する宣言(CSS プロパティ)です。
最小幅を設定する min-width と 最大幅を設定する max-width は、 width を上書きします。 


/* length 値 */
width: 250px;
width: 75rem;

/* percentage 値 */
width: 50%;

/* キーワード値 */
width: auto;
width: max-content;
width: min-content;

length 値
width: 250px;
width: 75rem;


CSS には、長さ (length) の値を表わす単位が数多くあります。

px は、画面表示における伝統的な単位です。
どのブラウザーでも 1px の長さは変わらない 絶対的な単位 です。

rem は デフォルトで設定されたフォントサイズ (多くのブラウザーでは既定値 16px) を 1 とする単位です。
ユーザーが独自にフォントサイズを設定している場合は、その設定に応じて 1rem の長さは異なる 相対的な単位 です。


length 値 で 幅を指定する際は、ウインドウサイズによって 要素がはみ出る可能性を考慮する必要があります。


percentage 値
width: 50%;


% は、親要素からの相対的な寸法を定義するためによく使われます。
このページでは、BODY（親要素）の中にある MAIN 要素の幅を操作するため、100%はBODYの幅と等倍を意味します。


/* max-width と width を併用する例 */

width: 750px;
max-width: 100%;

width: 100%;
max-width: 750px;

max-width を 100% に設定した場合、 例えば width: 750px; の要素は、ウインドウサイズが 750px 以下の場合に 幅 100% で表示されます。

逆に、width: 100%・max-width: 750px; と設定した場合は、ウインドウサイズが 750px 以下の場合は、幅 100% で表示され、 ウインドウサイズが 750px 以上の場合は、幅 750px で表示されます。


/* ~~~ */


要素全体の幅と高さをどのように計算するのかを設定する
box-sizing: content-box;
box-sizing: border-box;


すべての要素（CSS ボックスモデル）は既定で、 width および height を コンテンツ領域のみに適用する content-box が設定されています。
要素に境界やパディングがある場合、画面に表示される要素 は width/height の 値 + padding・border の 値 となるため、 例えば width: 100%; と指定しても border・padding が 設定されていれば画面から要素がはみ出てしまいます。


border-box は、 境界領域まで（コンテンツ領域 + padding・border）を width および height の値に含む設定です。
padding・border の値を配慮せず、簡単にサイズ設定ができるようになります。


/* ~~~ */


キーワード値
width: auto;
max-width: max-content;
min-width: min-content;


width の指定がない場合、 auto 要素の幅をブラウザーが計算します。


max-content は、望ましい固有の幅
min-content は、最小の固有の幅です。
固有の幅 は、ボックス内のテキストの長さなどから計算します。
例えば、max-content は、ボックス内のテキストを改行せずに横一列に表示した場合に 一番長いテキストの幅
min-content は、そのテキスト を 空白 などで文節として区切った場合に 一番長い文節 を 意味します。


改行（文節）の指定
word-break: normal;
word-break: break-all;
word-break: keep-all;


word-break の指定がなければ、 CJK (中国語、台湾語、日本語、韓国語) 以外のテキストの文節がコンテンツボックスからはみ出ても normal 改行されません。


break-all は、 文節にかかわらずテキストがコンテンツボックスに収まるように、強制的にどんな文字の間でも改行する設定です。
break-allの場合は 文節という概念がなくなるため、一番大きな幅の文字 が min-content となります。


keep-all は、文節を区分しずらいCJK (中国語、台湾語、日本語、韓国語) テキストの改行を許可しません。
CJK 以外のテキストについては normal と同じ挙動となります。

簡単な位置指定の例
margin: auto;
margin-left: auto;
margin-right: auto;


左右のマージンを auto に設定すると、幅を設定した子要素は親要素の中央に配置されます。
左右どちらかのマージン (margin-left・margin-right) を auto に設定した場合は、要素は ページの右端 または 左端に配置されます。


マージンの相殺

重なり合うブロックの上下のマージンは、それぞれのマージンの最も大きい寸法 (または等しい場合はいずれか 1 つ) の単一のマージンに結合されることがあります。
この動作を一般的に マージンの相殺 と呼びます。
