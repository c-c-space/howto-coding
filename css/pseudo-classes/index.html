    擬似要素 (pseudo-elements)

    選択された要素の特定の部分にスタイルを適用させるセレクターです。
    擬似クラスとは対照的に、擬似要素は要素の特定の部分スタイル付けを行うことができます。


    ~~~

    lightbox.css
    非表示にした .lightbox 要素を .lightbox:target 状態のときに表示する 

    .lightbox {
  position: fixed;
  color: #fff;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  pointer-events: none;
  user-select: none;
  opacity: 0;
  visibility: hidden;
  transition: 1s;
  z-index: 100;
}

.lightbox:target {
  width: 100%;
  height: 100%;
  pointer-events: auto;
  user-select: auto;
  opacity: 1;
  visibility: visible;
}

.lightbox .close,
.lightbox .close::after {
  display: block;
}

.lightbox .close {
  background-color: rgba(0, 0, 0, .75);
  color: #fff;
  transition: color 1s;
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  width: 100%;
  height: 100%;
}

.lightbox .close:hover {
  color: yellow;
}

.lightbox .close::after {
  content: "Close";
  font-size: 150%;
  position: absolute;
  right: 0.5rem;
  top: 0.5rem;
}

.lightbox,
.lightbox .close,
.lightbox .close::after {
  box-sizing: border-box;
  border: none;
  padding: 0;
  margin: 0;
}

    :empty

    要素の中身（コンテンツ）がない場合にスタイルを適用します。

    a:empty {
  color: #fff;
  background: deeppink;
  font-size: 200%;
}
    i am a

    上記4つのアンカー要素のうち、 a:empty が適用されるのは初めの1つのみです。
    2つ目にはテキスト、3つ目には半角スペース、4つ目には 空の em 要素があるため、a:empty ではありません。

    :not()

    否定擬似クラス (negation pseudo-class) 

    article:not(.readme) {
  background: #000;
}

:not(p):not(font) {
  background: silver;
}

body :not(#left):not(#right):not(#title):not(.readme) {
  box-shadow:
  -1px 1px 0 #000,
  -2px 2px 0 #000,
  -3px 3px 0 #000,
  -4px 4px 0 #000;
}

    :not() を使用する際のふつうではない効果や結果がいくつかありますので、使用する際には気を付けてください。
