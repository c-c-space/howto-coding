:empty

要素の中身（コンテンツ）がない場合にスタイルを適用します。

a:empty {
  color: #fff;
  background: deeppink;
}
i am a

上記4つのアンカー要素のうち、 a:empty が適用されるのは初めの1つのみです。
2つ目にはテキスト、3つ目には半角スペース、4つ目には 空の em 要素があるため、a:empty ではありません。


:not()

否定擬似クラス (negation pseudo-class) 

article:not(.readme) {
  background: #000;
}

:not(p):not(font) {
  background: silver;
}

body :not(#left):not(#right):not(#title):not(.readme) {
  box-shadow:
  -1px 1px 0 #000,
  -2px 2px 0 #000,
  -3px 3px 0 #000,
  -4px 4px 0 #000;
}

:not() を使用する際のふつうではない効果や結果がいくつかありますので、使用する際には気を付けてください。
