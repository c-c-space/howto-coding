フォントファミリー
font-family: キーワード値;


テキストのフォントを設定するには、font-family プロパティを使用します。

CSSで指定したフォントがユーザーのコンピューターで利用できない場合は、指定通りのフォントは適用されません。


~~~


5つの総称名

serif 明朝体
sans-serif ゴシック体
monospace 等幅フォント
cursive 筆記体
fantasy 装飾的なフォント


コンピューターのOS（Windows・Mac・Linux・Android・iOS・など）によって、インストールされているフォントは異なりますが、フォントファミリーを大きく区分する5つの総称は、すべてのOSで共通して利用できます。

serif・sans-serif・monospace は、ほぼすべての OS・ブラウザーで使用でき、OS・ブラウザーが妥当なフォントを提供するはずです。 ursive・fantasy は、慎重に使用してテストすることをお勧めします。


~~~


複数のフォントを指定する

フォントスタック


例)

font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
ブラウザーはリストの先頭からフォントが利用可能かを確認し、利用できるフォントを要素に適用します。


ファミリー名を , で区切り、最初に指定したフォントが利用できないコンピューターに代用するフォントを複数指定できます。

半角スペースを含むファミリー名は、ダブルコーテーション " で囲み、 フォントスタックの最後には , を付けず、; で終わる


フォントスタックは 同じ総称で統一し、最後に総称フォント名を指定することをお勧めします。


~~~


スタイル・太さ・変化形
font-style: キーワード値;
font-weight: 太さの値;
font-variant: キーワード値;


フォントファミリーには、いくつかの字体 バリエーション が含まれる場合があり、 font-style ・ font-weight ・ font-variant プロパティでスタイル・太さ・変化形を指定できます。


font-style: normal; 通常のフォント
font-style: italic; イタリック体
font-style: oblique; 通常のフォントを斜めにする


font-weight: normal; 通常の太さ
font-weight: bold; 太字
font-weight: 100 ~ 900; 太さを数値で指定する


font-variant: normal; 通常のフォントフェイス
font-variant: small-caps; スモールキャピタル (小ぶりの大文字) の字形を使用


フォントファミリーにイタリック体・スモールキャピタルが含まれない場合、italic・small-caps は適用されません。
太さの数値は 100 が最も細く、900が最も太い 値です。現実的には normal と bold 以外のものを使用することはほとんどありません。


~~~


weight style small-caps
font-synthesis

太字やイタリックの字体がない場合にブラウザーが合成してよいかどうかを制御する


none 字体を合成しないことを示すキーワード
weight 太字の字体を合成してよいことを示すキーワード
style イタリックの字体を合成してよいことを示すキーワード
small-caps スモールキャップの字体を合成してよいことを示すキーワード


weight style small-caps のように キーワード値を半角スペースで区切り、複数の合成を許可することもできます。

font-size は、フォントの大きさを定義するCSSプロパティです。

サイズの値には、絶対的なサイズを指す キーワード値 ・ px・rem などの単位を使う 数値 ・ 親要素のフォントサイズからの相対的なサイズを指す % があります。


~~~


キーワード値 と px

font-size: xx-small; 一部のブラウザーは最小フォントサイズが 10pxに設定されているため x-small と同等
font-size: x-small; 一般的には、10px と同等
font-size: small; 一般的には、13px と同等
font-size: medium; 一般的には、16px と同等
font-size: large; medium の 1.33倍 (18px)
font-size: x-large; large の 1.33倍 (24px)
font-size: xx-large; x-large の 1.33倍 (32px)
font-size: xxx-large; xx-large の 1.33倍 (42px)


rem と %

html {
  font-size: 16px;
}

main {
  font-size: 1rem;
}

main h1 {
  font-size: 200%;
}

サイズを表す数値の単位 rem は、HTMLやブラウザーの既定値を 1 とする 相対的な単位です。

上記のCSSでは、1rem は html のフォントサイズの値 16px です。
html をセレクターに フォントサイズの指定を行わなかった場合は、ブラウザー（ユーザー）が設定したフォントサイズが 1rem です。


% は、親要素のフォントサイズからの相対値です。

上記のCSSは、main要素内 の h1要素 のフォントサイズを、main要素のフォントサイズの2倍に指定します。
main のフォントサイズは 1rem（16px）なので、200% のフォントサイズは 2rem（32px）です。 



html {
  font-size: 90%;
}

body {
  font-size: 1rem;
}

上記のCSSは、ブラウザー（ユーザー）が設定するフォントサイズの 90% のサイズを 1rem とします。
すなわち、bodyのフォントサイズは、ブラウザーの既定値・ユーザーの独自設定によって相対的に変化します。



ビューポートのパーセント値による寸法

vw ウインドウの幅の 1% が 1vw
vw ウインドウの高さの 1% が 1vh
vmin vw と vh の小さい方の 1% が 1vmin
vmax vw と vh の大きい方の 1% が 1vmax


これらの単位は、ビューポート（ウインドウや印刷レイアウトのサイズ）に対する相対的な寸法です。

ウインドウサイズに関わらず、ページ全体の幅・高さの比率からサイズを適用する際に使用します。


~~~


font-size-adjust は、 (大文字の大きさを定義する) 現在のフォントサイズに相対的な小文字の大きさを設定するCSSプロパティです。


font-size-adjust に対応していないブラウザーと互換性を持たせるため、 font-size プロパティに掛け合わせる数値として指定します。

レイアウト
line-height: 高さの値;
letter-spacing: 幅の値;
word-spacing: 幅の値;
text-indent: 幅の値;


~~~


line-height は、 テキストの各ラインの高さを設定するCSSプロパティです。
一般的に見栄えがよく、読みやすいとされるラインの高さは約 1.5 〜 2（ダブルスペース）です。


letter-spacing は 各文字の字間 ・ word-spacing は タグや単語の間隔 を設定するCSSプロパティです。
幅の値には、px rem vw vh などの単位の数値を設定できます。


text-indent は ブロック内のテキストの行の前に置く空白 (字下げ) の幅を設定するCSSプロパティです。
幅の値には、px rem vw vh などの単位の数値、または % 値 を設定できます。

Google Fonts
Googleが無料で提供するフォントをページにリンクして使用することができます。


1. Google Fontsにアクセスし、好きなフォントを選択する

2. HTML コードをコピーして、フォントを使用する HTML文書 の HEAD要素に貼り付ける

3. 生成されたフォントファミリー宣言を、CSSに貼り付ける


例)
rel="preconnect" href="https://fonts.googleapis.com"
rel="preconnect" href="https://fonts.gstatic.com" crossorigin
href="https://fonts.googleapis.com/css2?family=Great+Vibes&display=swap" rel="stylesheet"


font-family: "Great Vibes", cursive;

~~~


@font-face {
font-family: "独自フォントのファミリー名";
src: url(独自フォントのURL) format("形式");
}


自身のコンピューターにインストールされたフォントを読み込む CSSルールセット

読み込み可能なフォントファイルの形式は、 "woff" "woff2" "truetype" "opentype" "embedded-opentype" "svg" です。


例)

@font-face {
font-family: "Bestermind";
src: url("BestermindRegular.ttf") format("truetype");
}


セレクター {

font-family: Bestermind, fantasy;

}
