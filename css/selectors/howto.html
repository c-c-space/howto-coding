CSS セレクター

CSS セレクターは、CSS のもっとも基本的なルールとなるものです。

CSS では、 ページ上の HTML 要素にスタイルを設定する ために セレクター を使用します。

CSS セレクター には さまざまな種類があり、 きめ細かにスタイルを設定する要素を選択できます。


universal selector
全称セレクター すべての種類の要素にスタイルを適用

* 
* {
  border: none;
  padding: 0;
  margin: 0;
}

以上の CSS ルールセット で、 すべての種類の要素 の 境界・余白 を なし(0) に スタイリングします。


type selector
要素型セレクター 要素名（ノード名）で要素を選択・スタイルを適用

要素名 
/* 要素型セレクターの例 */

html {
  height: -webkit-fill-available;
}

body {
  background: #000;
  min-height: -webkit-fill-available;
}

body,
a {
  color: #fff;
}

1つ目 の CSS ルールセット の、 セレクター は HTML要素

2つ目 の CSS ルールセット の、 セレクター は BODY要素

3つ目 の CSS ルールセット の、 セレクター は BODY要素 と A要素 です。
