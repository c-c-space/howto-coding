CSS の基本

CSS カスケーディングスタイルシート は、 HTMLの要素 を 選択的にスタイリングするために使う 宣言型 の スタイルシート言語 です。

CSS は、 セレクター (HTML要素の選択) と、 宣言 (要素をスタイル付けするための プロパティ と プロパティの値) の ルールセット で 構成されます。


セレクター

セレクターにはさまざまな種類があります。

また、セレクターをカンマで区切ることによって、複数の要素にルールセットを適用することもできます。


宣言 (プロパティ と プロパティの値)

例えば、見出し要素 で テキストを大きく見せたり、 リスト要素で箇条書きにしたり、 HTML だけでも、ページに基本的なスタイルを適用できますが、 それだけでは退屈なページになってしまいます。

Kill Styles このリンクをブックマークバーにドラッグしてください

閲覧しているページで、ブックマークバーに追加した Kill Styles をクリックすると、そのページにリンクしているすべての CSS が一時的に機能しなくなります。

まずは、もし CSS がなければ ページがどのように表示されるかを知り、CSSにできることを理解しましょう。
