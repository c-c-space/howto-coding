Multi-column layout
新聞のテキストの流れと同じように、コンテンツを複数の段ボックスにレイアウトする

columns
column-count (段の数)・column-width (段の幅) を一括指定する 

/* 段の幅 */
columns: 10rem;

/* 段数 */
columns: auto;
columns: 2;

/* 段の幅と段数の両方 */
columns: 2 auto;
columns: auto 10rem;
columns: auto auto;

column-gap
段同士の間隔 (段間)を指定する 

/* 初期値 (1em) */
column-gap: normal;

/* length 値 */
column-gap: 3px;
column-gap: 2.5em;

/* percentage 値 */
column-gap: 3%;

column-rule
段組みレイアウトで段間に引かれる線（段間罫）の太さ、スタイル、色を設定 

column-rule: dotted;
column-rule: solid 8px;
column-rule: solid blue;
column-rule: thick inset blue;

column-span
段組みレイアウトの段抜きを指定する 

/* 初期値 */
column-span: none;

/* すべての段にまたがらせる */
column-span: all;

column-fill
内容物が複数の段に分割されるとき、どのようにバランスを取るのかを指定する 

/* コンテンツを必要な余地があるときだけ分割 */
column-fill: auto;

/* コンテンツを各段に均等に分割 (初期値) */
column-fill: balance;

break-inside
ボックスの途中で、ページ、段、領域をどのように区切るかを設定する 

/* 初期値 */
break-inside: auto;

/* 主ボックスの途中に何らかの (ページ、段、領域の) 区切りを挿入することを禁止する */
break-inside: avoid;

/* 主ボックスの途中の改ページを禁止する */
break-inside: avoid-page;

/* 主ボックスの途中の段区切りを禁止する */
break-inside: avoid-column;

break-before break-after
ボックスの前後で、ページ、段、領域をどのように区切るかを設定する 

/* 一般の区切り値 */
break-after(before): auto;
break-after(before): avoid;
break-after(before): always;
break-after(before): all;

/* 改ページ値 */
break-after(before): avoid-page;
break-after(before): page;
break-after(before): left;
break-after(before): right;
break-after(before): recto;
break-after(before): verso;

/* 段区切り値 */
break-after(before): avoid-column;
break-after(before): column;

/* 領域区切り値 */
break-after(before): avoid-region;
break-after(before): region;
