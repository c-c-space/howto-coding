テキストの配置
text-align: キーワード値;
direction: キーワード値;
text-justify: キーワード値;


text-align は、ブロック要素（または表のセルボックス内）のインラインレベルコンテンツの水平方向の配置を設定する CSSプロパティです。


~~~


start ・ end

ブロックの書字方向に基づき配列を指定するキーワード値
書字方向 (direction) が左から右 (ltr) の場合、start は left ・ end は right と同じです。
書字方向 (direction) が右から左 (rtl) の場合、start は right ・ end は left と同じです。


left ・ right ・ center

書字方向に関係なく、インラインコンテンツの配列を指定するキーワード値
left は 左寄せ・right は 右寄せ・center は 中央寄せ に、コンテンツを配列します


justify

ブロック内のインラインコンテンツを両端揃えに配列するキーワード値
両端揃え text-align: justify; の種類は、text-justify プロパティで設定できます。


~~~


垂直方向の配置
vertical-align: キーワード値;


vertical-align は、インラインレベルコンテンツの垂直方向の配置を設定する CSSプロパティです。
vertical-align は インライン要素・インラインブロック要素・表のセル要素だけに適用できます。ブロックレベル要素には使用できません。


baseline ベースラインを親要素のベースラインに揃えるキーワード値
sub ベースラインを親要素の subscript-baseline に揃えるキーワード値
super ベースラインを親要素の superscript-baseline に揃えるキーワード値
text-top 要素の上端を親要素のフォントの上端に揃えるキーワード値
text-bottom 要素の下端を親要素のフォントの下端に揃えるキーワード値
middle 要素の中央を親要素の baseline + x-height の半分 に揃えるキーワード値
top 要素と子孫要素の上端を行全体の上端に揃えるキーワード値
bottom 要素と子孫要素の下端を行全体の下端に揃えるキーワード値

テキストの変換・装飾的な線
text-transform: キーワード値;
text-decoration: キーワード値;


~~~


text-transform は、テキストの大文字表記を指定する CSSプロパティです。


none すべての文字を変換させないキーワード値
capitalize それぞれの単語の最初の文字を強制的に大文字に変換させるキーワード値
uppercase すべての文字を大文字に変換させるキーワード値
lowercase すべての文字を小文字に変換させるキーワード値


~~~


text-decoration は、テキストの装飾的な線の 種類・色・太さ を一括指定する CSSプロパティです。

キーワード値を 半角スペースで区切り、複数の線を装飾することもできます。


none テキストの装飾を行わないキーワード値
underline テキストの各行に下線を引くキーワード値
overline テキストの各行の上線を引くキーワード値
line-through テキストの各行の中央を貫く線を引くキーワード値



次に、線を引く位置のキーワード値 に続いて、線の種類を指定します。

solid 単独線を描画するキーワード値
double 二重線を描画するキーワード値
dotted 点線を描画するキーワード値
dashed 破線を描画するキーワード値
wavy 波線を描画するキーワード値


線の種類を指定するキーワード値がない場合は、既定値の solid が設定されます。



色の値 太さの値


色を指定する値については CSS Colors 線の太さの値については Borders ページを参照ください。

文字に影を付ける
text-shadow: 水平 垂直 ぼかし 色;


~~~


text-shadow は、文字に影を付ける CSSプロパティです。


例)
4px 4px 5px crimson;
テキストシャドウ は 最大 4 つの値 で指定します。


1つ目の値は、水平方向への影の指定（値に - マイナスを付けると左方向へ影が移動します）
2つ目の値は、垂直方向への影の指定（値に - マイナスを付けると上方向へ影が移動します）
3つ目の値は、影をぼかす指定（省略可能。指定がなければ影はぼかされません）
4つ目の値は、影の色の指定（省略可能。指定がなければ color の値が適用されます）


~~~


文字に 複数の影 を付ける


例)
4px 4px 5px crimson,
skyblue -1vw 0 0.5vw,
0.25rem 0.25rem white;


影の値をコンマで区切って、同じ文字に複数の影を付けることができます。
色の値は、影の方向・ぼかしの値の前に記述することもできます。
