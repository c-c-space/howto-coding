背景画像
background プロパティの値に url("画像URL") を設定し、要素に1つ以上の背景画像を適用します。


複数の画像を背景に指定する場合は、url() を , (カンマ) で区切ります。
最初に記述した背景が一番上になり、後に記述した背景がその下になるように重ね合わされます。


~~~


background: 初期位置/サイズ url("画像URL");


初期位置を設定する値
top 画像を上辺揃えにします。
bottom 画像を下辺揃えにします。
left 画像を左辺揃えにします。
right 画像を右辺揃えにします。
center 画像を中央揃えにします。


サイズを設定する値
cover 画像を切り取ったり縦横比を崩したりすることなく、要素内で可能な限り大きくします。
contain 画像をコンテナーいっぱいにできるだけ大きく拡大縮小し、必要に応じて画像を引き伸ばします。
auto 背景画像の本来の比率が維持されるように、対応する方向に拡大縮小します。
幅 数値 または %値 で画像の幅を指定します。
幅 高さ 数値 または %値 を 半角スペースで区切り、幅 高さ の順にサイズを指定します


~~~


background: url("画像URL") 繰り返し 位置;


繰り返しを設定する値
repeat （既定値）画像は描画領域全体を覆うのに必要な回数だけ、繰り返し描画されます。
space 画像は指定された方向に切り抜きなしで敷き詰められる回数だけ繰り返されます。
round 利用できる空間が広がるにつれ、繰り返し描画された画像は、画像をもう1つ追加するだけの余裕 (残りの空間 ≧ 画像の幅の半分) ができるまで、 (隙間を空けずに) 伸長されます。
no-repeat 画像は繰り返し描画されません (したがって背景画像描画領域が完全に埋め尽くされるとは限りません)。
repeat-x 横方向にのみ、画像を繰り返し描画します。
repeat-y 縦方向にのみ、画像を繰り返し描画します。


位置を設定する値
scroll （既定値）背景は要素自身に対する相対位置で固定されます。
fixed 背景が固定され、スクロールしても動きません。
local スクロール可能なエリアを基準に、背景画像は要素の内容とともにスクロールします。


~~~


例)


/* 背景画像を 中央揃え/可能な限り大きく 繰り返さず 固定 する */

background {center/cover url(001.gif) no-repeat fixed}


/* 1つ目の画像を 左揃え/55pxサイズで 切り抜きなしで敷き詰められる回数だけ繰り返す */
/* 2つ目の画像を 右揃え/要素いっぱいにできるだけ大きく拡大縮小 全体を覆うのに必要な回数だけ繰り返す */

left/55px url(logo.png)　space, right/contain url(001.gif)
