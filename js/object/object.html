オブジェクト（key: value の組を持つ値）
let object = { key: value };

オブジェクトは、キー付されたさまざまなデータのコレクションや、より複雑な値を格納するために使用します。

{ key オブジェクト の キー : value キー の 値 , }

オブジェクトは、波括弧 { } とプロパティで構成されます。

プロパティ内のコロンの前がキー : 後が値（key:value）で、複数のプロパティを収納する場合は、プロパティをカンマ , で区切ります。


const docInfo = { // オブジェクト名
  'URL' : document.location,  // キー ['URL'] に値 document.location(このページのURL) が格納される
  'Last Modified' : document.lastModified,  // キー ['Last Modified'] に値 document.lastModified(このページの最終更新日) が格納される
  Referrer : document.referrer // キー .Referrer に値 document.referrer(リンク元のページのURL) が格納される
};

プロパティ値へは、オブジェクト名.プロパティキー または、オブジェクト名[プロパティキー] を使ってアクセスできます。

Object.keys(オブジェクト名)
指定されたオブジェクトが持つプロパティ名の配列を取得

Object.values(オブジェクト名)
指定されたオブジェクトが持つ列挙可能なプロパティの値を配列にして取得

Object.entries(オブジェクト名)
指定されたオブジェクトが持つプロパティの組(key, value)からなる配列を取得

オブジェクトのためのループ文を構成する
for...in for (let property in object)

object オブジェクト名

property 各ループ処理を宣言

let userDiv = document.querySelector ('#user')

for (let prop in user) { 

  userDiv.innerHTML += (`${prop} : ${user[prop]}`);

}

プロパティの短縮構文
実際のコードでは、既存の変数をプロパティ名の値として使用することがよくあります。 

function userInfo(device, hardware, user) {
  return {
    device, hardware, user
  };
}

let user = userInfo(
navigator.deviceMemory + ' GiB',
navigator.hardwareConcurrency + ' CPU',
navigator.userAgent
);

上の例では、プロパティは変数と同じ名前を持っています。変数からプロパティを作るユースケースでは、非常に一般的です。
