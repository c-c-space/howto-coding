順序付けされたコレクションを格納する配列宣言
let arr = [ 1, 2, 3 ];

角括弧 [ ] の中に配列の要素を、 カンマ , で区切り設定します。

arr[0] arr[1] arr[2]

配列にはゼロから始まる番号が付けら、角括弧にその番号を指定することで要素を取得できます。

arr.length

.length プロパティで 配列の要素の数を取得できます。

for...of
for (let key of arr)
反復可能オブジェクト（配列のためのループ文）を構成する

arr 配列名

key const・let・var で各ループ処理を宣言 


let itemArrAll = document.querySelectorAll('#popshift font');

for (let removeAll of itemArrAll) {
  removeAll.remove()
}

/* ~~~ */

forEach()
配列名.forEach(element => { })
配列の各要素に対して一度ずつ処理を実行する 


arr.forEach(key => {
  let codeArr = document.createElement('code');
  codeArr.textContent = key;

  listArr.appendChild(codeArr);
})

/* ~~~ */

pop/push shift/unshift
配列から要素を取り出す・追加するメソッド

arr 配列の末尾から要素を 取り出す

arr 配列の末尾に要素を 追加

arr 配列の先頭の要素を 取り出す

arr 配列の先頭に要素を 追加

メソッド push と unshift は一度に複数の要素を操作することができます
