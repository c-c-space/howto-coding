DOMツリーの中にある 要素の属性 を操作する

Element.attributes
要素に設定されたすべての生きた属性コレクションを返却するプロパティ

Element.hasAttributes
要素が属性を持っているか否かを示す真偽値を返すプロパティ

例) document.querySelectorAll で指定したすべての要素が hasAttribute 属性を持っているかを測定し、 attributes 持っていれば、属性コレクションを innerHTML すべて表示する 

const result = document.getElementById("result");

const attributes = document.querySelectorAll("article section");
for (const attribute of attributes) {

  if (attribute.hasAttributes()) {
    let attrs = attribute.attributes;
    let output = "";
    for(let i = attrs.length - 1; i >= 0; i--) {
      output += `${attrs[i].name}="${attrs[i].value}" `;
    }
    result.innerHTML += `<h3>${output}</h3>`;
  } else {
    result.innerHTML = "表示する属性はありません";
  }

}

Element.classList
クラスリストを操作するプロパティ

Element.classList.add()
Element.classList.remove()
Element.classList.replace()
Element.classList.toggle()

各メソッドを用いてオブジェクトを変更することはできます。

例) a[href^="#"] アンカーリンク先の article section のクラスリスト classList を link.hash によって add() 追加・ remove() 削除・ replace() 置き換える 

const anchorLinks = document.querySelectorAll('a[href^="#"]');

anchorLinks.forEach((link) => {
  link.addEventListener('click', function() {

    const readmeSectionAll = document.querySelectorAll("article section");
    for (const readmeSection of readmeSectionAll) {
      readmeSection.classList.remove("active");
      readmeSection.classList.add("is-hide");
    }

    const targetId = link.hash;
    const targetElement = document.querySelector(targetId);
    targetElement.classList.replace("is-hide", "active");
  });
});

Element.getAttribute(name)
指定した要素の指定された属性の値を返却するプロパティ

Element.removeAttribute(name)
指定した要素から指定した属性を削除するプロパティ

Element.setAttribute(name, value)
指定した要素に新しい属性を追加する・存在する属性の値を変更するプロパティ

Element.toggleAttribute(name, force)
指定した要素に対して属性をトグル切り替えするプロパティ（存在する場合は取り除き、存在しない場合は追加する）
