Element API
DOMツリーの中にある すべての要素 を 操作する API

文書自身の操作 には Document・文書を収めるウィンドウの操作 には Window API を使用します。

JavaScriptを実行する要素を指定するメソッド

Element.querySelector()
main・h1などの要素名・IDの値に一致する フラグメント (# 記号で記述) などから、1つの要素を指定します。

Element.querySelectorAll()
p・sectionなどの要素名、class 属性の値から要素を指定、セレクター名を , で区切り、複数の要素を指定します。

要素の内容（テキスト・コンテンツ・html）を操作するプロパティ

Element.innerText
要素の中にあるテキストの内容を表します。

.innerText と .textContentの違い
innerText は「人間が読める」テキストのみを示します。
textContent は、script・style 要素を含む、すべての要素の中身を取得します。

.textContent と .innerHTMLの違い
textContent は値が HTML として解析されません。
innerHTML は、その名が示すとおり HTMLタグを含めて値を返します。

要素を特定の箇所に挿入するメソッド

Element.before()
Element の直前に、(Node または DOMString) を挿入します。

Element.prepend()
Element の最初の子の前に、(Node または DOMString) を挿入します。

Element.append()
Element の最後の子の後に、(Node または DOMString) を挿入します。

Element.after()
Element の直後に、(Node または DOMString) を挿入します。

例) document.querySelector で指定した要素の innerHTML を変更し、 append を挿入する 

const titleH1 = document.querySelector('#cover h1');

titleH1.innerHTML = "<i>Document Object Model</i><br/>";
titleH1.append('Element');

/* ~~~ */

Element.insertAdjacentText()
テキストノードをDOMツリーの指定した位置に挿入します。

Element.insertAdjacentElement()
指定された要素ノードをDOMツリーの指定した位置に挿入します。

Element.insertAdjacentHTML()
テキストを HTML または XML として解釈し、解釈結果のノードをDOMツリーの指定した位置に挿入します。

以上3つのメソッドの引数(キーワード値, 値)

キーワード値
'beforebegin' 要素自体の前
'afterbegin' 要素の直下、最初の子の前
'beforeend' 要素の直下、最後の子の後
'afterend' 要素自体の後

キーワード値で設定した相対位置に、, カンマ に続く 値 を挿入します。

例) document.querySelector で指定した要素自体の後 afterend に insertAdjacentHTML の値を挿入する 

const titleH1 = document.querySelector('#cover h1');

titleH1.insertAdjacentHTML ( 'afterend', '<div class="log"></div>' );

/* ~~~ */

Element.outerHTML
要素とその子孫を含む部分の HTML がシリアライズされたものを取得します。
要素の内容のみの HTML 表現を取得する場合や、要素の内容を置き換える場合であれば、代わりに innerHTML プロパティを使用してください。

例) document.querySelector で指定した要素を outerHTML の内容を置き換える 

const titleBtn = document.querySelector('#cover button');

titleBtn.outerHTML = "<p id="p_log">Click on Double-click or Touch Here</p>";

/* ~~~ */

Element.remove()
指定された要素をDOMツリーから削除します。

例) #cover .log p をすべて remove 削除する 

let coverPAll = document.querySelectorAll('#cover .log p');

for (let coverP of coverPAll) {
  coverP.remove()
}
