要素の幅と高さを示すプロパティ
CSSOM width & height

Element.offsetWidth HTML要素のレイアウト幅（境界線、パディングを含む）
Element.clientWidth コンテンツの幅 (パディングを含み、境界、マージン、スクロールバーは含まない)
Element.scrollWidth 画面上に表示されない部分を含めた、要素の中身の幅
documentElement.clientWidth ルート要素のビューポートの幅(スクロールバーを除く)
documentElement.scrollWidth ルート要素全体の幅 (境界線、マージン、スクロールバーを除く)
Element.offsetHeight HTML要素の高さ（境界線、パディングを含む）
Element.clientHeight コンテンツの幅 (パディングを含み、境界、マージン、スクロールバーは含まない)
Element.scrollHeight 画面上に表示されない部分を含めた、要素の中身の高さ
documentElement.clientHeight ルート要素のビューポートの高さ(スクロールバーを除く)
documentElement.scrollHeight ルート要素全体の幅高さ (境界線、マージン、スクロールバーを除く)

Element.getBoundingClientRect()
要素の寸法と、そのビューポートに対する相対位置に関する情報を返すメソッド 

let clientRect = document.querySelector(".readme") // メソッドを実行する要素を指定

let rect = clientRect.getBoundingClientRect();
for (var key in rect) {
  if(typeof rect[key] !== 'function')  {
    let rectKey = document.createElement('h3');
    rectKey.textContent  = `${key} : ${rect[key]}`;
    document.querySelector("#rect").appendChild(rectKey);
  }
}
