TouchEvent
タッチ感応面（タッチ画面やトラックパッド）への接触状態によってイベントを実行

touchmove
タッチ感応面で 1 つ以上のタッチ点が移動したときに実行

EventTarget.addEventListener('touchmove', e => {
// JavaScriptを記述
});

EventTarget.ontouchmove = function() {
// JavaScriptを記述
}

touchstart
タッチ感応面に 1 つ以上のタッチ点が配置されたときに実行

EventTarget.addEventListener('touchstart', e => {
// JavaScriptを記述
});

EventTarget.ontouchstart = function() {
// JavaScriptを記述
}

touchend
タッチ感応面から 1 つ以上のタッチ点が取り除かれた場合に実行

EventTarget.addEventListener('touchend', e => {
// JavaScriptを記述
});

EventTarget.ontouchend = function() {
// JavaScriptを記述
}

touchcancel
１つ以上のタッチポイントが実装固有の方法で中断されたときに実行

EventTarget.addEventListener('touchcancel', e => {
// JavaScriptを記述
});

EventTarget.ontouchcancel = function() {
// JavaScriptを記述
}
