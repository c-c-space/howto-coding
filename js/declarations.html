何か面白いプログラミングをするには 変数・定数 を 宣言（定義） する必要があります。

変数・定数 を宣言することで、 それらの 値 を動的に操作し、ウェブサイトに対話性を追加することができます。

/* ~~~ */

変数は、データのための 名前付けされた格納場所 です。

let 変数名;

let というキーワードと、任意の名前（変数名）を指定して、変数を宣言します。
変数名には、文字・数字・$・_ のみが使用でき、最初の文字は数字であってはいけません。

let 変数名 = 値;

変数には、= (代入演算子) を使って、値（初期化子）を格納できます。
初期化子が設定されていない場合は、undefined という値が代入されます。

例)
let hello = 'Hello, ', // 変数名 hello に、値（文字列）Hello, を格納
userName = 'World'; // 変数名 userName に値（文字列）World を格納
 // 可読性のため、1変数1行にしてください。

let message = hello + userName + '!';

変数（変数名と値の組）を、, (カンマ) で区切り、複数の変数を宣言することもできます。
また、変数の値に変数名を使用して、他の変数の値を変数に代入 ・ + (加算演算子) を使って 変数の値 を結合することができます。
""・ ''・ ``・ (引用符・クォーテーション) で囲まれた値は、文字列と判断されるため、 誤って 変数名 を 引用符 で囲んだ場合は、値が代入されません。

/* ~~~ */

定数は、変更することが出来ない（変更されるべきでない） 読み取り専用の名前付き情報 です。

定数を作るには、 let の代わりに const キーワードを使います。

例)
const TITLE = ' | How to Coding',
titleH1 = document.querySelector('h1');

定数名のすべてを大文字にし、その値が決して変わらないことを強く示す慣習は広く行われています。
略語や短い名前は避け、分かりやすい名前を設定してください。

定数には、常に初期化子が必要です。 undefined で初期化することは、おそらくプログラマーのミスです。
