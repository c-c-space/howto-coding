JavaScriptの基本
JavaScript は、ウェブサイトに対話性を追加するプログラミング言語です。

ページの内容・スタイルを動的に操作したり、フォームへのデータ入力を行ったり、ウェブページを活かすために、ネットスケープコミュニケーションズのブレンダン・アイクによって開発されました。


HTMLにJavaScriptを適用する

src="ファイル名.js">
</body>

script要素をbody要素内に追加し、src属性の値に外部ファイルのパスを指定

type="text/javascript">
/* JavaScript を記述 */

JavaScriptを実行したい任意の部分にscript要素を挿入


JavaScriptを実行する
ジャバスクリプトファイル .js または、script要素内に、 宣言・値・関数（イベント）・条件文 などのコマンド(命令文)を記述して、JavaScriptを実行します。 

src="ファイル名.js" async>
type="text/javascript" defer>

"use strict"

/* JavaScript を記述 */

async と defer
script要素の開始タグに async または defer 属性を設定すると、 スクリプトのブロッキングの問題を回避できます。
ページのその他の部分（DOM など）と並行して、ブラウザーにスクリプトを別スレッドでダウンロードするよう指示するので、読み取りの過程で読み取りプロセスがブロックされることはなくなります。

async
ダウンロードが完了し次第、すぐに実行されます。 これはページをブロックし、特定の実行順序を保証することはありません。

defer
出現順に読み込まれ、すべての読み込みが終了した後に実行されます。

use strict
スクリプトの先頭に、"use strict" もしくは 'use strict' と記述があれば、 すべてのスクリプトは 最新の方法 で動作します。

// この行の // 以降がコメントです。

/*
コメントが複数行になる場合に / と * でコメントを囲みます。
*/

JavaScriptが複雑になった場合、何についての文なのかを説明するコメントを残すと便利です。
コメントは どの場所にも書くことができ、それらを無視するので実行には影響しません。
