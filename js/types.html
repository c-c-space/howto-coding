JavaScript の 値 には、 8つの基本的なデータ型 があります。

String（文字列）テキストの値を表す連続した文字
Number（数値）あらゆる種類の数値: 整数または浮動小数点
Boolean（論理値）true または false
undefined（未定義）値が未定義の未割り当て値
null 不明な値を意味する特殊なキーワード
BigInt（長整数）任意の長さの整数値のための型
Symbol（シンボル）一意な識別子

Object （オブジェクト）より複雑なデータ構造


引用符（クォーテーション） で囲まれた値は、 文字列 と呼ばれます。


" "ダブルクォート
' 'シングルクォート
` `バッククォート

ダブル/シングルクォートは、シンプルな引用符です。 JavaScriptの中ではそれらに違いはありません。

バッククォートは 拡張機能 の引用符です。
変数・定数を ${…} の中にラップすることで、変数・定数を埋め込み文字列の中で表現することができます。


数値 と *（乗算）・ /（除算）・ +（加算）・ -（減算） などの演算子を使えば、値の 計算 ができます。

+（加算） は、数値以外にも 文字列 などを結合することもできます。


論理値 boolean 型は、true と false の2つの値だけを持ちます。

論理値は通常 yes/no の値を格納するために使われます。
true は “yes” を意味し、 false は “no, 正しくない” を意味します。


未定義 初期化子が設定されていない変数には、undefined という値が代入されます。

定数には、常に初期化子が必要です。 undefined で初期化することは、おそらくプログラマーのミスです。

不明な値
nullは、 無し・空・または 不明な値 と言った意味を持つ特別な値です。

JavaScriptでは、null は他の言語のような “存在しないオブジェクトへの参照” または “null へのポインタ” ではありません。


長整数 const bigInt = 1234567890123456789012345678901234567890n;

BigInt の値は整数値の末尾に n を追加することで作成されます。
BigInt 型は任意の長さの整数を表現するために、最近言語に追加された型です。

オブジェクトとシンボル
object 型は特殊です。symbol 型はオブジェクトの一意な識別子を作るのに使われます。

他のすべての型は、値は1つのもの(文字列, 数値, または何でも)だけを含むので、“プリミティブ” と呼ばれます。
対照的に、オブジェクトはデータのコレクションやより複雑なエンティティを格納するために使われます。
