ループ文・反復可能オブジェクト
for と while

リストから1つずつコンテンツを出力するような場合や、1から10の数値それぞれに同じコードを実行する場合など、 複数回同じコードを繰り返す ループ文を作成します。

for (初期化式; 条件式; 加算式)
セミコロンで区切った 3 つの式と、実行する文からループ文を構成する

初期化式 ループを開始する最初の値を設定
let i = 0 // 変数 i の 値 を 0 に設定 (0 から ループをカウントする)

条件式 ループを実行する条件を設定
i <= 10 // i の 値 が 10(以下なり) になるまで ループを実行する

加算式 ループをカウントするステップを設定
i++ // インクリメント (++) を設定し、ループを実行するたびに i の値に 1 を加算
i-- // デクリメント (--) を設定し、ループを実行するたびに i の値に 1 を減算

変数 colors の値の数 li 要素を生成

i 0 から <= colors.length まで ++ 1 を加算し ループを実行

let colors = [

];

for (let i = 0; i <= colors.length; i++) {

  // ループ度に、この ブロック文 { ... } を実行

  let names = document.querySelector('ul#names');

  let list = document.createElement('li');
  list.textContent = colors[i];
  list.style.color = colors[i];

  names.appendChild(list);

}

.length プロパティで 条件式 の値を、配列の要素の数に設定
ループ度に、querySelector 選択した要素の appendChild 子ノードに、createElement li 要素を生成
子ノードの textContent と style.color の値に [i] ループごとの値を代入

/* ~~~ */

while (条件式)
条件式 が true の間、ループを実行

条件式 ループの条件を定義する式
n <= 3 // 変数 n の 値 が 3(以下なり) になるまで ループを実行する

let n = 0;

while (n <= 3) {

n++;

}

do...while do { 文 } while (条件式)
条件チェックをループ本体の 下に 移動

let n = 0;

do {

n += 1;

} while (n <= 3);

/* ~~~ */

for...of for (let key of arr)
反復可能オブジェクト（配列のためのループ文）を構成する

arr 配列名

key const・let・var で各ループ処理を宣言

let listArr = document.querySelector('#listArr');

for (let color of colors) { 

  let codeArr = document.createElement('code');
  codeArr.textContent = color;
  codeArr.style.color = color;

  listArr.appendChild(codeArr);

}
