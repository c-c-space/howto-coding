.Function (関数)
関数は、利用したい機能をパッケージ化する方法です。

関数名(alert・prompt・promptなど)を使って命令を呼び出したり、必要な処理をまとめて関数を定義(命令)・機能をパッケージ化することができます。
もし変数名に見えるものがあったとしても、その後に括弧 () が付いていれば、おそらくそれは関数です。

ユーザーと対話するためにダイアログを呼び出す関数

これらは、関数名と引数のみで命令を実行できる特殊な関数です。

alert(引数); 任意のメッセージを含むアラートを呼び出す関数

() の中に記述した引数が、メッセージとして表示され、OKがクリックされるとダイアログが閉じられます。

prompt(引数); メッセージ・入力フィールド・OK/CANCEL ボタンを持つダイアログを呼び出す関数

() の中に記述した引数が、メッセージとして表示され、, (カンマ) に続く次の引数が、入力フィールドの初期値として表示されます。
テキストが入力された場合は その値、未入力・キャンセル が押された場合は null の値を返します。

confirm(引数); 質問 と OK/CANCEL ボタンを持つモーダルウィンドウをダイアログを呼び出す関数

() の中に記述した引数が、質問として表示され、OK が押された場合は true、CANCEL の場合は false の値を返します。

関数は、 function と、それに続く 引数と定義 または、 => と、引数と定義 で構成されます。

// 関数式
function (引数) {定義(命令文)}
function キーワードは、式の中で関数を定義するために使用されます。

// アロー関数式
(引数) => {定義(命令文)}
アロー => キーワード は、従来の 関数式の簡潔な代替構文ですが、制限があり、すべての場面で使用することができるわけではできません。

引数(パラメーター) は、関数に渡される値(プリミティブまたはオブジェクト)です。 複数の引数がある場合はカンマ , で区切ります。

作成した関数を呼び出すために、 () の前に 関数名 を付けることができます。
function 関数名() { }

アロー関数式の場合は、変数・定数と組み合わせて 関数名を付けます。
let 関数名 = () => { }

ウェブサイトに対話性を追加するには、イベントが必要です。
イベントは、ブラウザーの中で起きていることを検出し、その応答としてコードを実行するコード構造です。

最も分かりやすい例は click イベント
要素がクリックされると、JavaScriptが実行されます。

EventTarget.addEventListener('click', 関数名)

.addEventListener() は、要素（ターゲット）にイベントを追加する関数です。

引数（パラメーター）の値に、('click', 関数名)を記述して、クリックイベントを設定します。

a・input・button などの要素は、onclick属性を使って click イベント を実行できます。
ダブルクリックでイベントを実行する場合は、ondblclick属性 を使用します。

要素をクリック（ダブルクリック）すると、 onclick・ ondblclick 属性の値に設定した、関数名() が、クリックイベントとして実行されます。

/* ~~~ */

EventTarget.addEventListener ('click', event => {

/* JavaScript を記述 */

})

.addEventListener の 引数（パラメーター）の値に 関数 を記述して、クリックイベントを設定することもできます。

/* 例) #count button を 連続してクリックした回数を取得する */ 

 type="text/javascript">

'use strict'

let click = document.querySelector('#count button')
let count = document.querySelector('#count font')

click.addEventListener('click', event => {
  count.textContent = event.detail
})

クリックとクリックの間隔が極めて短い場合、ダブルクリックやトリプルクリックのように扱われます。

/* ~~~ */

EventTarget.onclick = function() {

/* JavaScript を記述 */

}

その他にも、キーワード .onclick を使って、クリックイベントを実行できます。

/* 例) button#hello を クリックしたらアラートを呼び出す */ 

type="text/javascript">

'use strict'

const userName = 'World',
hello = "Hello, ";

const message = `${hello + userName}!`;


document.querySelector('button#hello').onclick = function() {
  alert( message ) ;
}
