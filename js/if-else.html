条件によって実行する命令文を定義する
if...else

if (条件式) {

// 条件が一致する場合に実行する文

} else {

// それ以外の場合に実行する文

}

if につづく () 内に、 比較演算子 を使って 条件式 を記述します。 

===  厳密な等価 (右辺と左辺が厳密に同一の値であるかを判定)
!==  厳密な不等 (右辺と左辺が厳密に同一の値ではないことを判定)
<  小なり (左辺の値が右辺の値より小さいこととを判定)
>  大なり (左辺の値が右辺の値より大きいことを判定)
<=  以下なり (左辺の値が右辺の値以下であることを判定)
>=  以上なり (左辺の値が右辺の値以上であることを判定)

else if キーワードを使って、 if...else に追加の条件を繋ぐ方法があります。
追加の選択肢は if() { ... } と else { ... } の間に記述します。

if (条件式) {

} else if (条件式) {

// 条件が一致する場合に実行する文

} else {

}

/* ~~~ */


confirm 関数で呼び出した 質問の回答が もし OK if (myAnswer === true) なら I'm OK! とアラートを表示し、 回答が それ以外(false) else なら I'm CANCEL! とアラートを表示する 

 type="text/javascript">

'use strict'

function yourAnsewer() {
  let myAnswer = confirm('Are you OK or CANCEL ?')
  if (myAnswer === true) {
    alert( "I'm OK!" )
  } else {
    alert( "I'm CANCEL!" )
  }
}

/* ~~~ */


prompt 関数で呼び出した 入力フィールドが もし キャンセル if (myAge === null) された場合は、 Top Secret! とアラート を表示し、 0以下 (未記入) else if (myAge <= 0) だった場合は、 In Utero とアラートを表示し、 それ以外 else の場合は、 その値を含むアラートを表示する 

 type="text/javascript">

'use strict'

function yourAge() {
  let myAge = prompt('How old are you?', 0)
  if (myAge === null) {
    alert( "Top Secret!" )
  } else if (myAge <= 0) {
    alert( 'In Utero' )
  } else {
    alert( `I'm ${myAge} years old!` )
  }
}
