Console API
デバッグ作業（メッセージ、コード内の設定点での変数の値、操作の完了にかかった時間などの記録）を実行できるようにする機能

使用方法は非常に簡単です — console オブジェクト — window.console またはワーカーの WorkerGlobalScope.console を介して利用できます。

console.log() (引数) に、出力するオブジェクトの「値」を設定
ウェブコンソールにメッセージを出力するメソッド
log()の引数は、String（文字列）、Array（配列）、Object（オブジェクト）、Boolean（真偽値）など、どのような型でも使用できます。

console.group()
console.groupEnd()
ウェブコンソールのログに、新たなインライングループを作成する
group()とgroupEnd()は、出力をブロックにグループ化し、インデントして表示します。

console.time()
console.timeEnd()
処理にかかる時間を計測する・呼び出しで開始したタイマーを停止する
処理時間を知りたいときに、この2つのメソッドを併用します。

console.table()
表形式のデータを表として表示するメソッド
配列やオブジェクトを自動的に表形式にします。

console.count()
実際に呼び出した回数を記録するメソッド
ループ内で使用して、特定の値が実行された回数を確認できます。

console.clear()
実行環境が許可すれば、コンソールを消去するメソッド
端末上に表示されているコンソールには、対応しておらず、何も効果がありません（エラーも発生しません）。

console.error()
エラーメッセージを出力するメソッド
デフォルトでは、エラーメッセージは赤色で強調表示されます。

console.warn()
警告メッセージを出力するメソッド
デフォルトでは、警告メッセージは黄色で強調表示されます。
