Window API
Window インターフェイスは、 DOM 文書を収めるウィンドウを表します。

document プロパティは、そのウィンドウに読み込まれた DOM の document オブジェクト を指します。
特定の文書が属するウィンドウは、 document.defaultView プロパティを使用して取得できます。

Window.navigator
スクリプトを実行しているアプリケーションに関するメソッドとプロパティを持つ Navigator オブジェクトへの参照を返します。

オンライン状態は true オフライン状態は false の論理値を返します。
navigator.onLine

const onLine = navigator.onLine

// 論理値の結果によって異なる命令を実行する //
document.querySelector('#onLine').innerText =
(onLine == true) ? 'オンライン状態' : 'オフライン状態'

Window: online イベント
addEventListener('online', (event) => { });
ononline = (event) => { };
Navigator.onLine の値が true(オンライン状態) に切り替わると発行されます。

Window: offline イベント
addEventListener('offline', (event) => { });
onoffline = (event) => { };
Navigator.onLine の値が false(オフライン状態) に切り替わったときに発行されます。

navigator.deviceMemory

Window.location
現在位置についての情報を持つ Location オブジェクトを返します。

URL 全体
location.href

URL のプロトコル
location.protocol

URL のドメイン
location.hostname

URL のオリジン
location.origin

URL のパス部分
location.pathname

location.assign(location.href)
引数で指定した URL のリソースを読み込みます。

location.reload()
現在の URL のリソースを再読み込みします。

location.replace(location.protocol + '//' + location.hostname + location.pathname)
現在のリソースを、指定した URL のリソースで置き換えます。

alert('URL: ' + location.toString())
これは Location.href と同義ですが、こちらは値を変更するために使用できません。

Window.history
履歴 API ブラウザーのセッション履歴へのアクセスを提供します。

前のページに戻る
() に設定した値のページに戻ります。 前のページがない場合、このメソッド呼び出しは何もしません。

次のページに進む
() に設定した値のページに進みます。 次のページがない場合、このメソッド呼び出しは何もしません。

セッション履歴から特定のページを読み込む
() に設定した値が 正数 + なら次のページに進みます。負数 - なら前のページに戻ります。
