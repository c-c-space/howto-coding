Navigator.onLine
Online or Offline?

インターネットの交信状態（オンライン/オフライン）を意味する論理値を返します。

オンライン状態の場合は true / オフライン状態の場合は false です。

body onload="update(navigator.onLine)" ononline="update(true)" onoffline="update(false)"

function update(online)

online ? 'true の場合' : 'false の場合'
オンライン状態に応じて宣言する文をパッケージ化 

function update(online) {
  document.querySelector('#status').textContent =
  online ? 'You are: Online' : 'You are: Offline';

  const onlineHideAll = document.querySelectorAll('.hide_online');
  for (const onlineHide of onlineHideAll) {
    onlineHide.style.display = online ? 'none' : 'block';
  }

  const offlineHideAll = document.querySelectorAll('.hide_offline');
  for (const offlineHide of offlineHideAll) {
    offlineHide.style.display = online ? 'block' : 'none';
  }
}

navigator.connection
Network Information API 
const connection = navigator.connection;

if (connection !== undefined) {
  const init = function() {
    document.getElementById('type').textContent = connection.type;
    document.getElementById('effectiveType').textContent = connection.effectiveType;
    document.getElementById('downlink').textContent = connection.downlink + ' Mb/s';
    document.getElementById('downlinkMax').textContent = connection.downlinkMax + ' Mb/s';
    document.getElementById('rtt').textContent = connection.rtt + ' ms';
  };

  init();

  if ('onchange' in connection) {
    connection.addEventListener('change', init);
  } else if ('ontypechange' in connection) {
    connection.addEventListener('typechange', init);
  }
}

ネットワーク情報 API は、システムのネットワーク接続に関する情報を提供する 実験的な機能 です。

この機能は、ユーザーの通信環境・接続状態に基づき、提供するデータを制限する場合などに便利です。


NetworkInformation.type ネットワーク接続の種類 (bluetooth・cellular・ethernet・none・wifi・wimax・other・unknown)
NetworkInformation.effectiveType ネットワーク接続の有効な種類 (slow-2g・2g・3g・4g)
NetworkInformation.downlink 回線の下り速度 (インターネット上からデータを受け取る際の通信)
NetworkInformation.downlinkMax 下り最大速度 | 1秒間に送受信可能なデータ量を表す単位 メガビット毎秒 (Mb/s)
NetworkInformation.rtt 推定有効ラウンドトリップ時間 (通信相手に信号を発信してから、応答が帰ってくるまでにかかる時間)
