Screen インターフェイスは、現在の画面のプロパティを検査する特別なオブジェクトです。
プロパティから画面のサイズや色深度・ピット深度などを取得できます。

Screen.width 画面の幅 (px)
Screen.availWidth タスクバーなどを除くスクリーン有効域の幅
Screen.height 画面の高さ (px)
Screen.availHeight タスクバーなどを除くスクリーン有効域の高さ

画面の色深度
Screen.colorDepth | bits per pixel

画面のビット深度
Screen.pixelDepth | bits per pixel

画面方向 API
Screen Orientation API 

// 現在の画面方向を取得する
let orientationType = (screen.orientation || {}).type || screen.mozOrientation || screen.msOrientation;

// 画面の向きに応じて宣言する文をパッケージ化
if (orientationType === "landscape-secondary" || orientationType === "landscape-primary") {
  document.getElementById("orientation").innerText = 'Type: ' + orientationType;
} else if (orientationType === "portrait-secondary" || orientationType === "portrait-primary") {
  document.getElementById("orientation").innerText = 'Type: ' + orientationType;
} else if (orientationType === undefined) {
  document.getElementById("orientation").innerText = "not supported by your browser";
}

landscape-primary 横幅が縦幅より大きい標準方位
landscape-secondary 横幅が縦幅より大きい標準方位と逆の向き
portrait-primary 縦幅が横幅より大きい標準方位
portrait-primary 縦幅が横幅より大きい標準方位と逆の向き
