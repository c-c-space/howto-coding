window.screen.width スクリーンサイズ(解像度)
window.screen.height スクリーンサイズ(解像度)
window.screen.availWidth タスクバーなどを除くスクリーン有効域の幅
window.screen.availHeight タスクバーなどを除くスクリーン有効域の高さ
window.innerWidth ブラウザ内の表示域(スクロールバーを含む)
window.outerWidth ブラウザ表示域(ブラウザ全体の外周)
window.innerHeight ブラウザ内の表示域(スクロールバーを含む)
window.outerHeight ブラウザ表示域(ブラウザ全体の外周)

/* ~~~ */

resizeTo(width, height)
width ウインドウの幅をピクセル数で表します。 height ウインドウの高さをピクセル数で表します。

resizeBy(xDelta, yDelta)
xDelta はウィンドウを水平方向に広げるピクセル数です。 yDelta はウィンドウを垂直方向に広げるピクセル数です。
