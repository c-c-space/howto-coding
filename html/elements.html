HTMLの歴史

1989年3月12日、「ウェブの父」として知られる イギリスの計算機科学者「ティム・バーナーズ＝リー」が WWW（ワールド・ワイド・ウェブ）の概念をまとめた 提案書 を発表した日が、インターネットの起源とされています。

世界初のウェブサイト info.cern.ch が誕生したのは、1991年8月6日。 スイスのアルプスにある欧州原子核研究機構（CERN）の施設で、36歳のティム・バーナーズ＝リーによって公開されました。

HTML 1.0

1993年6月、インターネット標準化団体 IETF の IIIR Working Group から Internet Draft として、HTML の素案 Hypertext Markup Language (HTML) が公開されました。 著者はティム・バーナーズ＝リーです。

HTML 2.0

1995年11月、IETF から正式に Hypertext Markup Language - 2.0 が公開され、BR(改行)、HR(水平線)、FORM(フォーム)などの要素が追加されました。
また、1996年5月には新たに TABLE関連要素 HTML Tables が追加されました。

1994年頃から、Firefox ブラウザの前身である Netscape 社 の Netscape Navigator ・ Microsoft社の Internet Explorerでも、 独自のHTML要素が開発され、ブラウザ戦争が繰り広げられます。

そんな中、1994年10月1日に ティム・バーナーズ＝リーは WWWの可能性を全て引き出すための共同事業体 「ワールド・ワイド・ウェブ・コンソーシアム」（略称「W3C」ダブリュースリーシー）を設立しました。

HTML 4.x

W3C は 1997年に HTML 4.0 Specification、 1999年に HTML 4.01 Specification を勧告しました。
また、2001年には日本語のルビに関する要素 Ruby Annotation を追加しました。

しかし、W3Cに対して不満を持つウェブ開発者・利用者が現れ、 2004年6月4日に 「Web Hypertext Application Technology Working Group」（略称「WHATWG」ワットワーキンググループ） が設立されました。
WHATWGは、Apple・Mozilla・Opera（後にMicrosoftも参加）の開発者たちによって結成されたHTMLの標準規格を作成する団体です。

その後、W3Cは 2007年3月に W3C HTML WG を発足して、WHATWG と共同作業を開始しますが、 WHATWGは、2011年から 版数の無い HTML Living Standard を開始し、2012年に共同作業は中止されました。
この頃からHTMLの標準は、W3C が勧告する規格と、WHATWG が策定を進める HTML Living Standard の 2つに分裂します。

HTML5

W3C は 2014年10月に HTML5、 2016年11月に HTML 5.1、 2017年12月に HTML 5.2 を独自に勧告しました。

2019年5月28日
W3C は独自の HTML の標準化を断念し、今後の HTML 標準化は WHATWG にゆだねることを決定。

そして 2021年1月28日、W3Cが定めていたHTML関連の仕様はすべて廃止され、HTML標準は WHATWG の HTML Living Standard に完全統一されました。
