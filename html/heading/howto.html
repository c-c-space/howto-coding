HTML の主要な仕事の一つは、ブラウザーが HTML 文書を意図通りに表示できるように、ウェブページに構造を与えることです。

ウェブページ内の文章は、見出し と 段落 で構成されます。
見出し要素には h1・h2・h3・h4・h5・h6 の 6つ のレベルがあり、各段落には p 要素を使用します。

P: 段落要素 については、別ページ テキストの基礎 を参照ください。

H1 - H6

H1 は、1 ページに 1 つだけ。ページタイトルなどの最上位の見出しに使用します。

見出し要素は正しい順序で使用するようにしてください。
例) H3 要素を使用した後に H2 要素を使用した場合は、見出し要素の意味を成しません。

1ページ につき使用する見出しレベルは 3 つまでにすることを目標としましょう。
多数のレベルがある文書は、閲覧者・コンピューターにとって扱いにくく、操作しにくくなるため、ページを分けることをお勧めします。

見出し要素 を使用した コンテンツ構成は、あらゆるユーザー・コンピューターにとって便利です。

区分要素（article・section など）に、その区分の内容を示す見出しを記述して、人間・コンピューターに識別できるようにすることが適切です。

検索エンジンは、見出し要素 を ページの検索ランクに影響する 重要なキーワード とみなします。
見出し要素がないページは、 SEO （検索エンジン最適化）の点で良くないでしょう。

極めて目が不自由な人は、画面朗読ソフトを使用してページを閲覧します。
ページの概要を見出し要素として用意することで、必要な情報へすばやくアクセスすることに繋がります。

視覚的にページレイアウトを構成するためでなく、コンテンツに正しい意味や機能を与えることが HTML の 大きな役割です。
