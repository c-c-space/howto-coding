文書の本文要素

BODY要素 は、HEAD要素に続く 2番目 の要素として、HTML に1つだけ配置できる要素で、 テキスト・画像・ビデオ・ゲーム・音声ファイルなど ブラウザに表示するすべてのコンテンツ を 記述します。

フローコンテンツ

HTML Living Standardでは、 BODY要素で使用される多くの要素を フローコンテンツ と分類します。
また、フローコンテンツは、用途に応じて 「区分要素」「見出し要素」「埋め込み要素」「対話的要素」「フォーム関連要素」 などに分類されます。


ononline="onFunction()" onoffline="offFunction()" >

BODY要素 には、インターネットの交信状態（オンライン/オフライン）によって呼び出す関数を設定できます。 

type="text/javascript">


function onFunction() {
  ここに オンライン状態の時に実行する処理 を記述する
}

function offFunction() {
  ここに オフライン状態の時に実行する処理 を記述する
}
