文書の本文要素

BODY要素 は、ウェブサイトとして ブラウザーに表示される 部分です。
HTML要素の中に、HEAD要素に続く 2番目 の要素として、一つだけ配置できます。

BODY要素 の中には、テキスト・画像・ビデオ・ゲーム・音声ファイルなど、 ブラウザーに表示するすべてのコンテンツ を 記述します。

フローコンテンツ

BODY要素で使用される多くの要素は フローコンテンツ と分類されます。
フローコンテンツは、用途に応じて 「区分要素」「見出し要素」「埋め込み要素」「対話的要素」「フォーム関連要素」 などに分類されます。


ononline="onFunction()" onoffline="offFunction()" >

BODY要素 には、インターネットの交信状態（オンライン/オフライン）によって呼び出す関数を設定できます。 

type="text/javascript">


function onFunction() {
  ここに オンライン状態の時に実行する処理 を記述する
}

function offFunction() {
  ここに オフライン状態の時に実行する処理 を記述する
}
