外部リソースへのリンク

HTMLに適用する CSS・サイトのアイコンなど、 現在の文書に関係する外部のリソースを指定します。
リンク要素のリンク種別属性(rel)に 外部リソースの種類を設定し、 href属性に外部リソースの場所を設定します。

HTML に CSS を 適用する

rel="stylesheet" href="styles.css"

/* スタイル情報要素 */

HEAD要素内の STYLE要素に、CSS を 記述
一般に、スタイルを外部スタイルシートに入れて LINK 要素を使用し適用する方法がより推奨されています。


rel="icon" href="ファビコン"

ファビコン・Webクリップアイコン（ホーム画面のタッチアイコン）など
もっとサイトデザインを豊かにするために、メタデータにカスタムアイコンを追加して、特定のコンテキストで表示されるようにすることができます。

必要に応じて BASE要素を設定し、 ページに記述されたリンク先 URL の基準となる URL・ 基準となる閲覧コンテキスト(ターゲットウィンドウ)を指定します。
