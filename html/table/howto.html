table tr td th

テーブルとは、行と列からなる構造化されたデータの集合 (表形式データ)です。

HTML の よくある仕事はテーブルデータの構造化で、数々の要素と属性がこの目的のためにあります。

テーブルの特徴は、厳密だということです。行と列とその見出し部分を組み合わせることによって情報を簡単に解釈できるため、正しく行われれば 視覚障害者でも晴眼者と同様に HTML テーブルの情報を解釈できるはずです。

HTMLテーブルは、table 要素を使用して作成します。
table 要素の中に tr 要素を記述して横並びの行（row）を 追加し、 tr 要素の中に td 要素を記述して縦並びの列（column）を 追加します。

テーブルのセル（マス目）が 表グループの見出しである場合は、 td 要素に代わって th 要素を使用します。

th要素 が 行または列 どちらの見出しなのかは、scope 属性で定義します。
scope="row" 横並びの行（row）の見出しである (セルが列の一番上にある) 場合
scope="col" 縦並びの列（column）の見出しである (セルが行の最初の列である) 場合

また、th (table headers) は td (table data) より目立つように、太字で中央に配置されます。

caption

HTMLテーブルに キャプション（またはタイトル）を指定する場合、 table 要素の中に caption 要素を記述します。

caption 要素 は、親である table 要素の最初の子とする必要があります。

caption 要素の位置指定には、CSS プロパティ caption-side を使用します。
caption-side: top; キャプションを表の上方に配置
caption-side: bottom; キャプションを表の下方に配置

td colspan="数値" td rowspan="数値"

td （またはtl）要素に colspan・rowspan 属性を設定すると、複数のセル（マス目）を一つに結合できます。

colspan="数値" 横に結合するセルの数を数値で設定
1000 を超える値は正しくないとみなされ、既定値 (1) が設定されます。

rowspan="数値" 縦に結合するセルの数を数値で設定
0 を設定した場合は、最後の行まで縦にセルが結合されます。

読み上げソフトのような支援技術は、colspan・rowspan属性を持つ表を解析するのが困難な場合があります。
支援技術を使用している人にも内容が理解しやすい表を作りたい場合は、colspan・rowspan属性に依存しないことを考慮してください。

id 属性と headers 属性

th要素に id 属性を設定し、関連するtd要素の headers 属性に同じ値を指定すると、セル間の関連付けを作成できます。

これをうまく機能させるためには、最初の列と最初の行の両方に表ヘッダー（th）要素が必要です。

テーブルに構造を追加する

表の構造が複雑になった場合、 thead tbody tfoot 要素を使って構造的な定義を与えることは有用です。

thead 列見出しを含む最初の行
tbody 表コンテンツの全ての行
tfoot 表を総括する最後の行

これらの要素によって、表が視覚障害者にとって理解しやすくなることはありませんが、視覚的な表のスタイリング・レイアウトに非常に役立ちます。

colgroup col

HTMLテーブルの最初の小要素 (最初の行 tr要素の 前) に colgroup・col 要素を記述し、 列全体のスタイル情報を 1 か所にまとめて定義することができます。

これらの要素によって、列ごとの背景色・幅などを簡単にスタイリングできます。
