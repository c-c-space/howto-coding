Attribute
<要素名 属性名="属性値">

要素を拡張して追加情報を与えるため、開始タグには 属性 を追加できます。

多くの属性は、 属性名="属性値" と設定しますが、中には属性名を記述するだけで設定できるものもあります。

属性には、すべての要素に設定できる グローバル属性 と、 特定の要素の機能・状態を指定する属性 の 2種類 があります。


代表的なグローバル属性

<要素名 id="値" class="値" >

ID は、要素に固有の名前を付けるグローバル属性
CLASS は、要素に共通の名前を付けるグローバル属性です。

<要素名 style="CSSルールセット" >

要素に適用する CSS スタイル宣言を包含します。なお、スタイルは別のファイルで定義することが推奨されます。

<要素名 lang="値" >

要素の言語の定義に使われます。 属性には単一の「言語タグ」を Tags for Identifying Languages (BCP47) で定義された書式で持ちます。

<要素名 contenteditable="値" >

ユーザーによる要素の編集が可能かを示す列挙型属性です。
要素が編集可能であることを示す値は true・ 要素が編集不可であることを示す値は false です。

<要素名 draggable="値" >

要素がブラウザーの本来の動作と HTML ドラッグ＆ドロップ API のどちらの動作でドラッグするかを示す列挙型属性です。
ドラッグ可能であることを示す値は true・ ドラッグ不可であることを示す値は false です。

<要素名 spellcheck="値" >

要素でスペルミスのチェックを行うかを定義する列挙型属性です。
スペルチェックを行うことを示す値は true・ スペルチェックを行わないことを示す値は false です。

<要素名 hidden>

要素はまだ、あるいはもはや関連性がないことを示す論理型属性です。
ブラウザーは、 hidden 属性を設定している要素を表示しません。
