要素名 class="値"
クラス属性は、主にスタイルシートのセレクターとして使用します。

同じ CLASS の値を 複数の要素 に使用して、共通する要素を選択したり、 値を 半角スペース で区切り、1つの要素 に 複数の値 を設定することもできます。

CLASSの値は、アルファベットで始めなければなりません。 また、 半角英数字・ - で設定することが推奨されています。

class selector
クラスセレクター class 属性の値から要素を選択してスタイルを適用

.CLASSの値 
.special {
  background: #000;
  color: red;
}

.super {
  font-style: italic;
  background: yellow;
}

.great {
  font-size: 150%;
}

.super.special {
  display: block;
}

.special .special {
  color: green;
}

.great .great {
  color: blue;
}
