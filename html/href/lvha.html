:link — :visited — :hover — :active 

a {
  text-decoration: underline;
}

a:link {
  color: #0000EE;
}

a:visited {
  color: #551A8B;
}

a:hover {
  cursor: pointer;
}

a:active {
  color: #FF0000;
}

アンカー要素には、デフォルトで以上のCSS擬似クラスが設定されています。


:link まだアクセスしていないリンクにスタイルを適用
:visited アクセスしたことのあるリンクにスタイルを適用

擬似クラス link・visited で、リンク先へのアクセスの有無によって 色を #0000EE ・ #551A8B に区別する

この 2つ の擬似クラスは、href 属性を持つ要素に使用できます。


:hover 要素にマウスオーバーしたときにスタイルを適用

擬似クラス hover で、マウスポインターが要素の上にいるときに表示されるマウスカーソルの種類を pointer に設定

擬似クラス :hover は、ほぼすべての要素に使用できますが、タッチ画面などのブラウザーではうまく機能しない場合があります。


:active 要素がアクティブ化されたときにスタイルを適用

擬似クラス active で、クリック中の要素の色を #FF0000 に指定

擬似クラス :active は、a・button 要素 に 設定されています。


これら 4つ の擬似クラスは、:link — :visited — :hover — :active の順に定義するようにしてください。
順番が守らなければ、CSS 宣言 が上書きされ、スタイリングが適用されません。
