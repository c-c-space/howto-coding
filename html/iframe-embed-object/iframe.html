iframe インラインフレームを設定する

YouTubeの動画や、Google Mapの地図などをウェブページに埋め込む際などに、 iframe 要素を有用します。

既定では、ブラウザーは iframe を境界線付きで表示します。
CSS で border: none を設定すると、境界線なしで表示されます。

width と height 属性を追加すると、iframe の 幅と高さを指定できます。
CSSプロパティ width と height を使うと、より詳細な指定ができます。 

src="フレーム内に表示する文書のURL"
name="閲覧コンテキスト名"
loading="lazy"
referrerpolicy="no-referrer"
srcdoc="インラインフレームの内容を属性値として記述する"
sandbox>

・閲覧コンテキスト名 name は、アンダースコア（ _ ）で開始されない任意の文字列で指定します。

・loading="lazy" を追加すると、レイジーロード（遅延読み込み）を実装できるようになります。
これは、文書の表示速度の改善や転送量の軽減に役立ちます。

・referrerpolicy="no-referrer" を追加すると、リファラーが送信されなくなります。
1つ前に閲覧していたページのURL/情報をリファラーと言います。

・srcdoc 属性に記述した値（HTMLコード）は、src属性より優先し、インラインフレーム内に埋め込まれます。
srcdoc属性に対応していないブラウザーでは、フレームに何も表示されません。 または、src属性 の URL で代替されます。

・sandbox フレーム内のコンテンツに追加の制約を適用します。
攻撃者がウェブサイトで悪事を働く力をできるだけ小さくしたいので、埋め込みコンテンツには「その仕事をするために必要な権限」だけを与えるべきです。
