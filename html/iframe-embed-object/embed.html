embed 文書にプラグインデータを埋め込む

Java アプレットや Flash のようなプラグイン技術が普及した 90 年代後半から 2000 年代前半に embed は、とても有用なものでしたが、現在ではほとんど使用されていません。 

<embed
  src="フレーム内に表示する文書のURL"
  type="埋め込むコンテンツのMIMEタイプ"
  title="埋め込むコンテンツの題名">

・embed 要素は、終了タグ </embed> の省略が可能です。

最近のほとんどのブラウザーは、ブラウザーのプラグインの対応を非推奨にして削除しているため、 embed に頼ることは賢明ではないということを意識しておいてください。
