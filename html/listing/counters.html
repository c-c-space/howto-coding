カウンターの使用
counter-reset: カウンター名 初期値;
カウンターを使用するには、最初に必ず counter-reset プロパティで値を初期化する必要があります。

増加または減少
counter-increment: カウンター名 カウンタに加える値;
カウンタに加える値の既定値は 1 です。数値に - をつければ減少カウントが設定できます。

カウンターの表示
counter: "文字" counter(カウンター名, リストスタイル名) "文字";
contentには任意のテキスト文字を設定できます。ラテン文字以外は Unicode エスケープシーケンスでエンコードする必要があります。 

/* id="this" 要素内の section 要素にカウンターを使用する */

#this {
  counter-reset: sections;
}

#this section::before {
  counter-increment: sections;
  content: "第 " counter( sections, cjk-ideographic ) " 章: ";
}

カウンタースタイル
CSS Counter Styles
独自のカウンタースタイル (リストのマーク) を定義する 

/* @counter-style 独自のカウンタースタイルを定義する */

@counter-style スタイル名 {

system: キーワード値;
symbols: "カウンター記号";
prefix: "カウンター記号の前に表示する記号";
suffix: "カウンター記号の後ろに表示する記号";

range: "スタイルが適用されるカウンター値の範囲";
negative: "負の値の場合の表現";

pad: 最小限の数値 "symbol";
additive-symbols: integer [0,∞] symbol;

speak-as: キーワード値;

fallback: 代替リストスタイル;

}
