順序なしリスト要素
Unordered List
一般的に、行頭記号を伴うリストとして描画されます。

順序付きリスト要素
Ordered List
reversed 属性を追加すると、リスト項目が逆順で番号付けされます。
属性 start="数値" を設定して、リスト項目の開始値を指定できます。

リスト子要素
List Item
順序付きリスト・順序なしリストの項目を表す小要素として使用します。

list-style-position:
リストのマーカー (::marker) を li の内に含むか、外に描画するかを指定する CSSプロパティ

リスト項目のマーカーを選択するCSS擬似要素
li::marker {

    font-size: 111%;
    font-style: italic;
    font-family: serif;

}

padding: margin:
リストのインデント
リスト要素にはデフォルトでインデント(パディングとマージン) が設定されています。
