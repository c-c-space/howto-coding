説明リスト

The Description List
説明用語要素 DT と 詳細説明要素 DD を 説明リスト要素 DL に記述して説明リストを表します。
Description Term
Description Details
dt
dl の小要素として、dd (または直接関連するdt) の前に使用できます。
dd
dl の小要素として、dt (または直接関連するdd) の後に使用できます。
一般的な使用例
用語集の作成やメタデータ（キーと値のペアのリスト）の表示
ディスプレイレベル display: block;
インデント margin: 値;
dl・dt・dd要素はすべて blockレベル要素
dl要素にはデフォルトで margin-top と margin-bottom 、dd要素にはデフォルトで インデント(margin-left) が設定されています。
説明を見やすくレイアウトするために、dd 要素に margin-bottom を設定する傾向があります。

/* ~~~ */

type="text/javascript">

const dlAll = {
  '用語 1' : '説明 1';
  '用語 2' : '説明 2';
};

let descriptionList = document.querySelector('dl#description');
for (let dtdd in dlAll) {
  descriptionList.innerHTML += (`<dt>${dtdd}</dt><dd>${dlAll[dtdd]}</dd>`);
}

定数 dlAll の オブジェクト値（key: value の組を持つ値）から ループ文(for...in) を構成し、 説明リスト dl#description の中に dt・dd 要素を生成するDOM操作を行う JavaScript
