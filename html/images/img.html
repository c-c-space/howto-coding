img 画像をウェブページに埋め込む

ウェブでよく使われる画像ファイル形式

APNG (Animated Portable Network Graphics)
可逆性のあるアニメーションシーケンス（ブラウザーの対応が狭いのが現状です。）

AVIF (AV1 Image File Format)
高い性能による、画像とアニメーションの両方

GIF (Graphics Interchange Format)
シンプルな画像やアニメーション

JPEG (Joint Photographic Expert Group image)
静止画の非可逆圧縮（現在最も普及しています。）

PNG (Portable Network Graphics)
JPEG より若干画質が良い（透明度が必要な場合には JPEG より好まれます。）

SVG (Scalable Vector Graphics)
ベクター画像形式（異なる大きさで正確に描画する必要がある画像などに最適です。）

WebP (Web Picture format)
画像とアニメーションの両方に最適

WebP や AVIF などは、PNG、JPEG、GIF よりはるかに性能が良く、静止画と動画の両方で使えるのでおすすめします。 WebP は広く対応されていますが、 AVIF は Safari での対応が不足しています。

src="画像のURL（必須）" alt="代替テキスト"

IMG 要素は、テキストコンテンツを含まないため、 終了タグ がありません。

・src 属性 に、埋め込みたい画像のURLを設定
URLは、相対 URL または絶対 URL にすることができます。

・alt 属性 に、画像が見えない/表示されない場合に使用する 代替テキスト (テキストによる画像の説明文) を設定
必須ではありませんがアクセシビリティのために非常に有用です。
