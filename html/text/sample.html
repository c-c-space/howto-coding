HTML の主要な仕事の一つは、ブラウザーが HTML 文書を意図通りに表示できるように、ウェブページに構造を与えることです。

これはテキストの段落を表す p 要素 です。p 要素 は ブロックレベル要素で、隣接するブロックレベル要素は新しい行に配置されます。

コンテンツをP要素で分割すると、読み上げソフトやその他の支援技術が前後の段落に飛ぶショートカットを提供するなど、ページのアクセシビリティを向上するのに役立ちます。
段落間の間隔を広げるために空の段落を使用することは、読み上げ技術の支援の下で操作をしている人にとっては問題になります。

デフォルトでは、ブロックレベル要素の幅は親要素と同等で、高さはコンテンツと同じになります。 また、ブロックの上下にはマージンが設定されます。
段落間の間隔の調整は、 margin のような CSS プロパティを使用して効果を得るようにしてください。

や のような インライン要素は 同じ行に、スペースがある場合は隣接して配置されます。

インライン要素の幅が、それを内包するブロックの幅をはみ出てオーバーフローする場合 は、適当な箇所で新しいラインに折り返されます。
