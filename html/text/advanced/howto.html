テキストの基礎 に書かれた HTML の他にも、テキストの書式設定のための多くの要素があります。

連絡先のマークアップ
<address>

HTML が 連絡先 (住所、 URL、メールアドレス、電話番号、ソーシャルメディアのアカウント、地理上の座標など) を提供していることを示します。 

a[href^='mailto']::before {
  content: '📧 ';
}

a[href^='tel']::before {
  content: '📞 ';
}

日付と時刻をマークアップ
<time>

時間と日付を機械可読形式でマークアップするための要素
datetime 属性を使用して、機械可読な形式の日付を記述します。

2018-07-07"> グレゴリオ暦の正確な日付
20:00"> 24 時間制の時刻
2018-07-07T20:00:00"> 日付と時刻

この要素は、機械可読な形式で日付や時刻を表現するためのものです。
例えばユーザーエージェントが、ユーザーのカレンダーにイベントの予定情報を提供することに役立てることができます。


関心のある文字列のマーク
<mark>

周囲の文脈の中で目立たせたり強調したりする文字列
興味のある部分が見つかったときに、蛍光ペンを使ってマークするようなものだと考えてください。

マークの背景色は、CSS background プロパティで指定できます。 
mark {
  background: yellow;
}

ブロック引用
<blockquote>

段落・複数の段落・リストなど、ブロックレベルでテキストを引用する
cite 属性の中に引用元を指す URL を記載します。

インライン引用
<q>

段落区切りをまたがない短い引用
多くのブラウザーでは、文字列を引用符で囲むように実装します。 

<blockquote cite="引用元URL">

<q>インライン引用</q>

<cite>引用元のタイトル</cite>

</blockquote>
インライン引用要素の引用符は、擬似クラス ::before ::after で指定できます。 
q::before {
  content: "☯️ ";
}

q::after {
  content: " ☯️";
}

文書への編集を表す
<ins> <del>

文書に追加されたテキストの範囲・文書から削除された文字列の範囲を表します。
例えば、「変更の追跡」や、ソースコードの差分情報を描画するときに使用します。

打ち消し線を引いた文字列
<s>

要素はすでに適切または正確ではなくなった事柄を表現します。


文書の修正を示す場合は del と ins 文章に打ち消し線を描写する場合は s が適切です。
追加された文字列・削除済み文字列をスタイリングする CSS 
del {
  text-decoration: line-through;
  background-color: #fbb;
  color: #555;
}

ins {
  text-decoration: none;
  background-color: #d4fcbc;
}

~~~

コンピューターのコードを表現する
<code> 行内コード要素
<pre> 整形済みテキスト要素
<var> 変数要素
<kbd> キーボード入力要素
<samp> サンプル出力要素

これらは、HTMLにコンピュータのコードをマークアップするために利用できる要素です。
例えば、HTML 内に CSSやJavaScript・HTMLコードを文字列として表示する際に使用します。

コンテンツの断片を機械可読な翻訳にリンク
値">内容

value属性の値に、要素の内容を機械可読な形式で指定します。
例えば、商品にそれぞれの商品番号も結びつけるなどの際に使用します。
