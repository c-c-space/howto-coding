track 埋め込みテキストトラック要素

自動的に処理される字幕など、時間指定されたテキストトラックを指定します。

default デフォルトのトラックリストを指定
この属性はメディア要素ごとに 1つの track 要素のみで使用できます。

kind どのように使用するトラックであるか
subtitles 字幕・翻訳
captions 音声の翻訳・非言語情報
descriptions テキストによるコンテンツの説明
chapters チャプタータイトル
metadata トラック情報 (ユーザーには見えません。)

label テキストトラックのタイトル
テキストトラックを一覧表示する際にブラウザーが使用する、ユーザーに見せるタイトルです。

track要素は、audio および video の小要素として使用できます。 


 default
src="ファイル名.vtt"
srclang="言語タグ"
kind="キーワード値"
label="ラベル名"

TRACK 要素の src 属性に、テキストトラックに使用する vtt 形式のWebVTTファイルのURLを指定し、 srclang 属性の値に、テキストトラックの言語を示す言語タグを設定します。
