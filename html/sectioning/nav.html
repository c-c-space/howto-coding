ナビゲーションセクション要素

一般的に、メニュー・目次・索引など、主要なリンクメニューを提供する区分にナビゲーションセクション要素を使用します。

すべてのリンクを nav に入れる必要はありません。サイト・ページの構成に関わるリンクメニューがナビゲーションに該当すると思われます。


例)
このページでは、ページ上部に固定しているリンクメニューに nav を使用しています。



#one">
#two">
#three">
#four">


:target

アンカーリンクをクリックして対象の要素へ移動したとき、その要素にスタイルを適用します。

#one:target,
#two:target,
#three:target,
#four:target {
  color: #fff;
  background: #FF0000;
}

擬似クラス :target は、一般的にアンカー要素 と 合わせて使用します。
上記のセレクターは、href属性 に #one・#two・#three・#four が設定された a要素 がクリックされ、同じ idの値 を持つ要素に移動した状態を選択します。
