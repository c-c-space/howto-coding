汎用セクション要素

このページには、異なるIDの値を持つ 4つの section があります。

id="one">
id="two">
id="three">
id="four">

通常、汎用セクション要素 は 親要素の中に 見出し (ヘディング要素) と合わせて 小区分 を作成する際に使用します。


そのコンテンツが小区分ではなく、独立した記事を示す場合は article・ メインコンテンツの余談的な内容を示す場合は aside が適切です。

要素名 id="値"
ID属性の用途は、リンク・CSS・スクリプト で 要素を特定することです。

すべての要素 は、ID の値を ひとつだけ 持つことができます。

IDの値は、 半角英数字・ _・ -・ . で設定することが推奨されています。
空白やタブなどのホワイトスペース文字を値に含むことはできません。

※ 同一ページ内に、同じ IDの値 を 使わないようにしましょう。


ID selector
IDセレクター 特定のIDの値を指す フラグメント (# 記号で記述) からスタイルを適用

#IDの値 
#one,
#three {
  color: #fff;
  background: blue;
}

#two,
#four {
  color: #000;
  background: #fff;
}

これは、 id="one" と id="three" の 色 を #fff(白)・背景 を blur(青色) に、 id="two" と id="four" の 色 を #000(黒)・背景 を #fff(白) に スタイリングする CSS ルールセットです。


リンク
#IDの値 ハイパーリンク先に特定のIDの値を指す フラグメント (# 記号で記述) を指定する

#one">
#two">
#three">
#four">


scroll-behavior
ハイパーリンク先の特定のIDへのスクロールの振る舞いを設定します。

html {
scroll-behavior: smooth;
}

smooth は、円滑にスクロールする設定です。
既定値の auto は、ターゲット先に瞬時にスクロールする設定です。
