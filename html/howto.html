HTMLは、ウェブサイトのコンテンツの構造を作るために使うマークアップ言語です。
Tag (タグ) を使って Element (要素) を作り（マークアップし）、要素の集まりがウェブサイトを構成します。

DOCTYPE宣言
すべての HTML文書は、このファイルが HTML だということをコンピュータに宣言するコードから始まります。

<html lang="ja">

これは、 HTML要素 の始まりを宣言する 開始タグ です。
HTMLにおいて、タグは要素の作成に使われます。
開始タグ (Opening tag): < html > のように山括弧の中に要素名を記述し、要素の始まりを宣言します。

属性 (Attribute)
要素を拡張して追加情報を与えるため、開始タグには 属性名="属性値" を追加できます。
HTML要素には、文書の言語を与える LANG属性 lang="言語タグ" を設定することが推奨されています。

HTML要素は、 HEAD要素 と BODY要素 で構成されます。

HTML の中で <!-- と --> の間にある コメント は ページ上では表示されず、コードの中にのみ表示されます。

HEAD要素 は、ページが読み込まれてもウェブブラウザーには表示されない部分です。

HTMLについてよい基礎を与える情報を記述し、HTMLを正しく表示する設定を行います。

BODY要素 は、ウェブサイトとしてブラウザーに表示される部分です。

BODY要素 の中に、テキスト・画像・ビデオ・ゲーム・音声ファイルなど、ブラウザーに表示するすべてのコンテンツを記述します。

</html>

これは、 HTML要素 の終わりを宣言する 終了タグ です。
要素は開始タグ、内容、終了タグが続いたものです。
終了タグ (Closing tag): </html> のように、 要素名の前にスラッシュを記載した終了タグで要素の終わりを宣言し、 開始タグと終了タグの中に、内容 (Content)を記述します。
