フォームへのスタイル設定
How to Styling Forms

<style type="text/css">

CSS を使用してフォームコントロールのスタイルを設定する方法
一部の要素は、CSS でスタイルを設定できません。

Basic
Font Styling

一部のブラウザーは、フォームコントロール要素に font-family や font-size を継承しません。

button, input, select, textarea {
font-family : inherit;
font-size : 100%;
appearance : none;
}

以上のルールセットで、親要素とコントロールのフォント・文字サイズを同じように設定します。
inherit は、親要素のプロパティ値を継承するキーワード値です。同じく 100% も、親要素の数値を継承する値です。

appearance プロパティは OS やシステムレベルでウェブフォームのスタイル設定を制御する方法として作成されました。
appearance: none; を適用したコントロールは システムレベルのスタイルを使用することを可能な限り止め、CSS を使用して自分でスタイルを構築できるようにします。

The Box Model

各コントロールは、境界・パディング・マージンについて独自のルールを持っています。

button, input, select, textarea {
box-sizing : border-box;
border : 0.1rem solid;
}

さまざまなコントロールを同じサイズにしたい場合は、box-sizing を border-box に設定します。
加えて、各コントロールの border の 太さを統一すると綺麗にスタイリングできます。

input[type="text"],
input[type="email"],
input[type="url"],
input[type="tel"],
input[type="password"],
textarea {
border-radius : 0.25rem;
padding : 0.5rem;
margin : 0.25rem 0;
width : 100%;
}

textarea {
min-width : 100%;
max-width : 100%;
max-height : 10rem;
max-height : 50vh;
}

テキストを入力するタイプのコントロールの width（height）・padding・margin・border を 指定します。
textarea に min・maxプロパティを設定すると、コントロールサイズの変更を制限できます。

フォーム関連要素 fieldset・legend も、境界・パディング・マージンについて独自のルールを持っています。
フィールドセット・キャプションをスタイリングする場合、box-sizing を border-box に設定すると便利です。

fieldset, legend {
box-sizing : border-box;
border : 0.2rem double;
}

fieldset {
border-radius : 1rem;
padding : 1rem;
margin : 1rem 0;
}

legend {
border-radius : 0.25rem;
font-size : 75%;
padding : 0.25rem 0.5rem;
text-align : center;
}

legend の位置は、text-alignで指定します。

Fieldset・Legend

フォーム関連要素 fieldset・legend も、境界・パディング・マージンについて独自のルールを持っています。
フィールドセット・キャプションをスタイリングする場合、box-sizing を border-box に設定すると便利です。
横方向での legend の位置指定は、text-alignを使用します。

fieldset, legend {
box-sizing : border-box;
border : 0.2rem double;
}

fieldset {
border-radius : 1rem;
padding : 1rem;
margin : 1rem 0;
}

legend {
border-radius : 0.25rem;
font-size : 75%;
padding : 0.25rem 0.5rem;
text-align : center;
}

/* ~~~ */

legend を fieldset の上境界の上以外に配置する場合はpositionを使用します。

fieldset {
position : relative;
}

legend {
position : absolute;
bottom : 0;
right : 0;
}

そのほかにも、transform プロパティを使用して legend の位置を指定する方法も一般的です。

Advanced
UI pseudo-classes

フォームコントロールの状態によって要素を選択できる擬似クラス

:hover
マウスポインターを当てた状態のコントローラーを選択します。

button:hover,
input:hover,
section:hover,
textarea:hover,
.checkbox label:hover,
.radio label:hover {
box-shadow : 0 0 0 0.2rem;
}

:focus
フォーカスされている状態のコントローラーを選択します。

input:focus,
select:focus,
textarea:focus {
outline : solid red;
}

:active
アクティブ状態（クリックされたとき、またはキーボード操作の場合は Return/Enter キーが押されたとき）のコントローラーを選択します。

button:active,
input:active,
select:active,
textarea:active {
transform : scale(1.1);
}

:enabled
すべての有効なコントローラーを選択します。

:disabled
disabled 属性が設定されている無効なコントローラーを選択します。

:empty
子を持たない空のコントローラーを選択します。

button:enabled,
input:enabled,
select:enabled,
textarea:enabled {
background : #fff;
}

input:disabled,
textarea:disabled {
opacity : 0.5;
}

input:empty,
textarea:empty {
color : #000;
}

:required
required 属性が設定されているコントローラーを選択します。

:optional
required 属性が設定されていないコントローラーを選択します。

input:required,
textarea:required {
color : red;
background : #fff;
font-weight : bold;
}

input:optional,
textarea:optional {
font-weight : lighter;
}

:valid
フォーム検証の結果、値が有効な状態のコントローラーを選択します。

:invalid
フォーム検証の結果、値が無効な状態のコントローラーを選択します。

input:valid {
filter : invert(0);
}

input:invalid {
filter : invert(1);
}

Checkboxes & Radio Buttons

チェックボックスを fieldset class="checkbox"、 ラジオボタンを fieldset class="radio" で囲み、 関連する label をスタイリングします。
input[type="checkbox"]・input[type="radio"] を非表示にします。

.checkbox input[type="checkbox"],
.radio input[type="radio"] {
display : none;
}

.checkbox label と .radio label を スタイリングします。
それぞれの境界の丸みを指定して見た目を区別します。

.checkbox label,
.radio label {
box-sizing : border-box;
border : 0.1rem solid;
background : #fff;
color : #000;
display : inline-block;
margin : 0.125rem 0;
}

.checkbox label {
border-radius : 0.25rem;
padding : 0.25rem 0.5rem;
}

.radio label {
border-radius : 1.5rem;
padding : 0.25rem 0.75rem;
}

/* ~~~ */

擬似クラス :default+label を使って、checked 属性があるチェックボックス・ラジオボタンをスタイリングします。

.checkbox input[type="checkbox"]:default+label,
.radio input[type="radio"]:default+label {
font-weight : bold;
}

擬似クラス :checked+label を使って、選択状態のチェックボックス・ラジオボタンをスタイリングします。

.checkbox input[type="checkbox"]:checked+label,
.radio input[type="radio"]:checked+label {
filter : invert(1);
}

input[type="checkbox"]・input[type="radio"] が選択状態のとき、直後にある label の 色を反転します。
